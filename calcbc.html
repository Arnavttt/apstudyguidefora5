Unit 1
Limits & Continuity
~10–12% of exam

Limit Laws & Evaluation Strategies

Direct Substitution

lim[x→a] f(x) = f(a) if continuous

Try this first. If you get 0/0 or ∞/∞, use another technique.

L'Hôpital's Rule (BC)

lim f(x)/g(x) [0/0 or ∞/∞] = lim f'(x)/g'(x)

Differentiate numerator and denominator separately. Can apply repeatedly.

Squeeze Theorem

g(x) ≤ f(x) ≤ h(x) lim g = lim h = L → lim f = L

Classic use: lim[x→0] x²sin(1/x) = 0

Special Trig Limits

lim[x→0] sin(x)/x = 1 lim[x→0] (1-cos x)/x = 0 lim[x→0] (1+x)^(1/x) = e

Continuity 3 conditions must ALL hold

f is continuous at x = a if ALL THREE hold:
1. f(a) is defined (a is in the domain)
2. lim[x→a] f(x) exists (left limit = right limit)
3. lim[x→a] f(x) = f(a)

Types of discontinuity:
Removable: hole — lim exists but ≠ f(a), or f(a) undefined
Jump: left limit ≠ right limit (step function)
Infinite: vertical asymptote (lim = ±∞)

Intermediate Value Theorem (IVT)

IVT — Highly Tested

If f is continuous on [a,b] and k is between f(a) and f(b), then there exists at least one c in (a,b) such that f(c) = k.

Exam use: "Show that f has a zero on [1,3]" — compute f(1) and f(3), show one is positive and one negative, cite continuity + IVT.

Units 2–3
Differentiation Rules
~10–12% of exam

All Derivative Rules

Power Rule

d/dx [xⁿ] = nxⁿ⁻¹d/dx [x⁵] = 5x⁴

Product Rule

d/dx [uv] = u'v + uv'

"derivative of first times second plus first times derivative of second"

Quotient Rule

d/dx [u/v] = (u'v − uv') / v²

Low d-high minus high d-low, over low squared. Never forget the minus!

Chain Rule

d/dx [f(g(x))] = f'(g(x))·g'(x)

Derivative of outside (keep inside) times derivative of inside.

Trig Derivatives

sin x → cos xcos x → −sin xtan x → sec²xsec x → sec x tan xcsc x → −csc x cot xcot x → −csc²x

Exponential & Log

d/dx [eˣ] = eˣd/dx [aˣ] = aˣ ln(a)d/dx [ln x] = 1/xd/dx [log_a x] = 1/(x ln a)

Inverse Trig (BC)

d/dx [arcsin x] = 1/√(1−x²)d/dx [arctan x] = 1/(1+x²)d/dx [arcsec x] = 1/(|x|√(x²−1))

Implicit Differentiation

x² + y² = 252x + 2y(dy/dx) = 0dy/dx = −x/y

Treat y as a function of x; every y-term gets a dy/dx factor.

⚡ Chain Rule Trap
  • d/dx [sin(3x²)] = cos(3x²) · 6x — don't forget the inner derivative!
  • d/dx [e^(x²)] = e^(x²) · 2x
  • d/dx [(x³+1)⁵] = 5(x³+1)⁴ · 3x² = 15x²(x³+1)⁴
Unit 5
Applications of Derivatives
~15–18% of exam

Critical Points, Extrema, Concavity

FINDING EXTREMA — 3-step process:
1. Find f'(x). Set f'(x) = 0 or undefined → critical points
2. First Derivative Test: f' changes + to − → local MAX; − to + → local MIN
3. Second Derivative Test: f''(c) < 0 → local MAX; f''(c) > 0 → local MIN; f''(c) = 0 → inconclusive

CONCAVITY:
f'' > 0 → concave up (holds water, ∪)
f'' < 0 → concave down (sheds water, ∩)
f'' changes sign → inflection point

Mean Value Theorem (MVT) & Rolle's Theorem

Mean Value Theorem

f'(c) = [f(b)−f(a)] / (b−a)

If f is continuous on [a,b] and differentiable on (a,b), there exists c in (a,b) where the instantaneous rate = average rate.

Rolle's Theorem

f(a) = f(b) → ∃c: f'(c) = 0

Special case of MVT. If f(a)=f(b), there's a horizontal tangent somewhere between a and b.

Related Rates & Optimization

RELATED RATES — always:
1. Draw a diagram. Label quantities.
2. Write an equation relating them.
3. Differentiate implicitly with respect to t.
4. Substitute known values and solve.

Example: Balloon radius r increases at dr/dt = 2 cm/s. Find dV/dt when r = 5.
V = (4/3)πr³ → dV/dt = 4πr² · dr/dt = 4π(25)(2) = 200π cm³/s

L'Hôpital's Rule (BC)

L'Hôpital — Key Points
  • ONLY valid for indeterminate forms: 0/0, ∞/∞, 0·∞, ∞−∞, 0⁰, 1^∞, ∞⁰
  • For 0·∞: rewrite as fraction (0/(1/∞) or ∞/(1/0)) to get 0/0 or ∞/∞
  • For 1^∞ or 0⁰: take ln, apply L'H, then exponentiate the result
  • Can apply repeatedly if still indeterminate after first application
Units 6–7
Integration
~17–20% of exam

Fundamental Theorem of Calculus

FTC Part 1

d/dx [∫ₐˣ f(t) dt] = f(x)

The derivative of an integral (with variable upper limit) is the integrand evaluated at the upper limit.

FTC Part 2

∫ₐᵇ f(x) dx = F(b) − F(a)

F is any antiderivative of f. Evaluate at upper bound minus lower bound.

Chain Rule + FTC1

d/dx [∫ₐ^g(x) f(t) dt] = f(g(x))·g'(x)

When upper limit is a function, multiply by its derivative. Very common FRQ part.

Integration Techniques

U-SUBSTITUTION:
∫ f(g(x))g'(x) dx — let u = g(x), du = g'(x) dx
Example: ∫ 2x·e^(x²) dx → u = x², du = 2x dx → ∫eᵘ du = eᵘ + C = e^(x²) + C

INTEGRATION BY PARTS (BC):
∫ u dv = uv − ∫ v du
Choose u and dv using LIATE: Logarithm, Inverse trig, Algebraic, Trig, Exponential
Example: ∫ x·eˣ dx → u=x, dv=eˣdx → uv−∫vdu = xeˣ − eˣ + C

PARTIAL FRACTIONS (BC):
∫ 1/(x²−1) dx → 1/[(x−1)(x+1)] = A/(x−1) + B/(x+1)
Solve: A=1/2, B=−1/2 → (1/2)ln|x−1| − (1/2)ln|x+1| + C

Key Antiderivatives to Memorize

Basic Antiderivatives

∫xⁿ dx = xⁿ⁺¹/(n+1) + C (n≠−1)∫1/x dx = ln|x| + C∫eˣ dx = eˣ + C∫aˣ dx = aˣ/ln(a) + C

Trig Antiderivatives

∫sin x dx = −cos x + C∫cos x dx = sin x + C∫sec²x dx = tan x + C∫sec x tan x dx = sec x + C

Inverse Trig Antiderivatives

∫1/√(1−x²) dx = arcsin x + C∫1/(1+x²) dx = arctan x + C∫1/(x√(x²−1)) dx = arcsec|x| + C
Unit 8
Applications of Integration
~10–15% of exam

Area, Volume, Accumulation

Area Between Curves

A = ∫ₐᵇ [top − bottom] dx A = ∫ₐᵇ [right − left] dy

Find intersection points for limits. Sketch to identify top/bottom.

Disk Method (solids)

V = π∫ₐᵇ [f(x)]² dx

Rotating around x-axis. Each cross section is a disk with radius f(x).

Washer Method

V = π∫ₐᵇ ([R(x)]²−[r(x)]²) dx

R = outer radius, r = inner radius. Use when there's a hole.

Shell Method (BC)

V = 2π∫ₐᵇ x·f(x) dx

Rotating around y-axis. Useful when disk/washer gives complicated integrals.

Arc Length (BC)

L = ∫ₐᵇ √(1 + [f'(x)]²) dx

Average Value

f_avg = (1/(b−a)) ∫ₐᵇ f(x) dx

Mean Value Theorem for Integrals: equals f(c) for some c in [a,b].

Unit 9
Differential Equations
~6–12% of exam

Separable Differential Equations

STEPS — Separable DEs:
1. Separate variables: get all y's on one side, all x's on other
2. Integrate both sides
3. Solve for y (or leave implicitly if asked)
4. Apply initial condition to find C

Example: dy/dx = xy, y(0) = 3
dy/y = x dx
ln|y| = x²/2 + C
|y| = e^(x²/2 + C) = Ae^(x²/2)
y(0)=3: 3 = Ae⁰ → A = 3
y = 3e^(x²/2)

Slope Fields & Euler's Method (BC)

Slope Fields

dy/dx = f(x,y) gives slope at each point. Draw short line segments to visualize solution curves. Exam asks you to match a slope field to a DE or sketch a solution through a given point.

Euler's Method (BC)

yₙ₊₁ = yₙ + h·f(xₙ, yₙ)

h = step size. Approximate y values numerically. Always show work in a table: x, y, dy/dx, Δy.

Logistic Growth (BC)

dP/dt = kP(1 − P/M) P(t) = M/(1+Ae^(−kt))

M = carrying capacity. Fastest growth when P = M/2. Inflection point there.

Taylor & Maclaurin Series — Must Memorize These 5

MACLAURIN SERIES (Taylor centered at 0):

eˣ = Σ xⁿ/n! = 1 + x + x²/2! + x³/3! + ... (all x)
sin x = Σ (−1)ⁿx^(2n+1)/(2n+1)! = x − x³/3! + x⁵/5! − ... (all x)
cos x = Σ (−1)ⁿx^(2n)/(2n)! = 1 − x²/2! + x⁴/4! − ... (all x)
1/(1−x) = Σ xⁿ = 1 + x + x² + x³ + ... (|x| < 1)
ln(1+x) = Σ (−1)ⁿ⁺¹xⁿ/n = x − x²/2 + x³/3 − ... (−1 < x ≤ 1)

TAYLOR SERIES centered at a:
f(x) = Σ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ
= f(a) + f'(a)(x−a) + f''(a)(x−a)²/2! + f'''(a)(x−a)³/3! + ...

Radius & Interval of Convergence

STEPS — always use Ratio Test:
1. Apply Ratio Test: find lim |aₙ₊₁/aₙ| = L·|x−a|
2. Set L·|x−a| < 1 → solve for |x−a| < R (where R = 1/L = radius)
3. CHECK ENDPOINTS separately using other tests (AT, p-series, etc.)
4. Write interval of convergence with correct open/closed brackets

Common result: converges at one endpoint, diverges at other (half-open interval)
⚡ Alternating Series Error Bound

|Error| ≤ |first omitted term|. If using first 3 terms of an alternating series, error ≤ |4th term|. Endpoints must still be checked even after finding the radius.

Unit 9 (BC) — BC Only ★
Parametric, Polar & Vector Functions
~11–12% of BC exam

Parametric Equations

dy/dx from parametric

dy/dx = (dy/dt) / (dx/dt)

Both are derivatives with respect to the parameter t.

d²y/dx² from parametric

d²y/dx² = [d/dt(dy/dx)] / (dx/dt)

Take derivative of dy/dx w.r.t. t, then divide by dx/dt again.

Arc Length (parametric)

L = ∫ₐᵇ √[(dx/dt)²+(dy/dt)²] dt

Speed (vector)

speed = √[(dx/dt)² + (dy/dt)²] = |v(t)|

Polar Coordinates

Conversions

x = r cos θ, y = r sin θ r² = x² + y² tan θ = y/x

Area in Polar

A = (1/2)∫ₐᵇ [r(θ)]² dθ

Area between two polar curves: (1/2)∫[r_outer² − r_inner²] dθ

Common Polar Curves

Circle: r = a Cardioid: r = a(1±cosθ) Rose: r = a cos(nθ) — 2n petals if n even, n petals if odd Lemniscate: r² = a²cos(2θ)

Practice
MCQ Practice
Q1Limits · IVT

f is continuous on [1, 5]. f(1) = −3 and f(5) = 7. Which of the following must be true?

  • A) f has no zeros on (1,5)
  • B) f has at least one zero on (1,5)
  • C) f has exactly one zero on (1,5)
  • D) IVT does not apply here
Answer: B — IVT: f continuous on [1,5], f(1)=−3 < 0 < 7 = f(5), so ∃c ∈ (1,5) with f(c) = 0. Not unique — there could be more.
Q2Derivatives · Chain Rule

If f(x) = sin(x²), what is f'(x)?

  • A) cos(x²)
  • B) 2x cos(x²)
  • C) −2x sin(x²)
  • D) −2x cos(x²)
Answer: B — Chain rule: outer = sin(u), inner = u = x². d/dx[sin(x²)] = cos(x²)·(2x) = 2x·cos(x²)
Q3Series · Convergence

Which of the following series CONVERGES?

  • A) Σ 1/n
  • B) Σ n/(n+1)
  • C) Σ 1/n²
  • D) Σ 2ⁿ
Answer: C — Σ1/n² is a p-series with p=2 > 1, so it converges. (It equals π²/6 — the Basel problem.) The others: A=harmonic(diverges), B=nth term=1≠0(diverges), D=geometric r=2(diverges).
Q4Integration · FTC

If g(x) = ∫₀ˣ sin(t²) dt, what is g'(x)?

  • A) cos(x²)
  • B) sin(x²)
  • C) 2x·sin(x²)
  • D) 2x·cos(x²)
Answer: B — FTC Part 1: d/dx[∫₀ˣ f(t) dt] = f(x). Here f(t) = sin(t²), so g'(x) = sin(x²). The upper limit is just x (no chain rule needed).
Q5Taylor Series

The Maclaurin series for eˣ is 1 + x + x²/2! + x³/3! + ⋯. What is the Maclaurin series for e^(2x)?

  • A) 1 + 2x + 2x²/2! + ⋯
  • B) 1 + 2x + 4x²/2! + 8x³/3! + ⋯
  • C) 1 + 2x + 2x²/2! + 4x³/3! + ⋯
  • D) 2(1 + x + x²/2! + ⋯)
Answer: B — Substitute 2x into eˣ series: Σ(2x)ⁿ/n! = 1 + 2x + (4x²)/2! + (8x³)/3! + ⋯ = Σ 2ⁿxⁿ/n!

FRQ Exam Structure
  • Section II: 6 FRQs (90 min). FRQ 1–3: calculator allowed. FRQ 4–6: no calculator.
  • Show ALL work — answers without supporting work earn no credit.

FRQ 1Area, Volume, and Accumulation

Let f(x) = 3 − x and g(x) = x² − 1.

PART A Find the x-coordinates of all intersection points of f and g.

PART B Find the area of the region enclosed between the two curves.

PART C Find the volume of the solid formed when this region is rotated about the x-axis.
PART A: Find intersections
3 − x = x² − 1
x² + x − 4 = 0
x = (−1 ± √17)/2 ≈ −2.56, 1.56
x = (−1−√17)/2 and x = (−1+√17)/2

PART B: Area
On the interval, f(x) ≥ g(x) (check: f(0)=3 > g(0)=−1)
A = ∫[(−1−√17)/2 to (−1+√17)/2] [(3−x) − (x²−1)] dx
= ∫ [4 − x − x²] dx
= [4x − x²/2 − x³/3] evaluated at bounds
A = 17√17/6

PART C: Volume (Washer Method)
Outer radius R = f(x) = 3−x, Inner radius r = g(x) = x²−1
V = π∫[(3−x)² − (x²−1)²] dx over same bounds
FRQ 2Series — Taylor Polynomial + Error

Let f(x) = cos(x).

PART A Write the 4th-degree Maclaurin polynomial P₄(x) for f.
PART B Use P₄(0.2) to approximate cos(0.2). Show all work.
PART C Use the Alternating Series Error Bound to find an upper bound for the error in part B.
PART A:
cos x = 1 − x²/2! + x⁴/4! − ⋯
P₄(x) = 1 − x²/2 + x⁴/24

PART B:
P₄(0.2) = 1 − (0.2)²/2 + (0.2)⁴/24
= 1 − 0.04/2 + 0.0016/24
= 1 − 0.02 + 0.0000667
≈ 0.980067

PART C: Alternating Series Error Bound
The first omitted term is x⁶/6! = (0.2)⁶/720
= 0.000000064/720 ≈ 8.9 × 10⁻¹¹
Error ≤ (0.2)⁶/720 ≈ 8.9 × 10⁻¹¹
(Actual cos(0.2) ≈ 0.980067, confirming excellent approximation)

⚡ Perfect Score Checklist

  • L'Hôpital requires 0/0 or ∞/∞ — verify the indeterminate form before applying
  • IVT requires continuity — always state "f is continuous on [a,b]" in your justification
  • FTC Part 1 + Chain Rule: d/dx[∫ₐ^g(x) f(t)dt] = f(g(x))·g'(x)
  • Series: always check endpoints separately after finding radius of convergence
  • nth-Term Test can only prove divergence, never convergence
  • Harmonic series diverges: Σ1/n → p=1 is NOT enough for convergence
  • Taylor polynomial vs. Taylor series: polynomial is finite (P₃, P₅ etc.), series is infinite
  • Polar area formula has a 1/2: A = (1/2)∫r²dθ
  • Euler's Method: always underestimates for concave up, overestimates for concave down
  • Logistic model inflection point: P = M/2 (half of carrying capacity)
  • Show units on accumulation problems: if rate is mph, integral gives miles
  • Justify continuity/differentiability before applying theorems
