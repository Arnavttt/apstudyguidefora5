Q1
The formal definition of a limit states that lim_{x→a} f(x) = L means
  • A) f(x) equals L when x is very close to a
  • B) the derivative at x = a equals L
  • C) for every ε > 0, there exists δ > 0 such that 0 < |x−a| < δ implies |f(x)−L| < ε
  • D) f(a) = L exactly
C is correct. for every ε > 0, there exists δ > 0 such that 0 < |x−a| < δ implies |f(x)−L| < ε
Q2
L'Hôpital's Rule applies when evaluating a limit that produces
  • A) the derivative of a product of functions
  • B) an infinite series
  • C) an indeterminate form (0/0 or ∞/∞), allowing differentiation of numerator and denominator separately
  • D) any rational function at a real number
C is correct. an indeterminate form (0/0 or ∞/∞), allowing differentiation of numerator and denominator separately
Q3
The Intermediate Value Theorem guarantees that if f is continuous on [a, b] and N is between f(a) and f(b), then
  • A) the derivative is zero somewhere on (a, b)
  • B) there exists at least one c in (a, b) where f(c) = N
  • C) f(a) = f(b) for all continuous functions
  • D) the function has a maximum on [a, b]
B is correct. there exists at least one c in (a, b) where f(c) = N
Q4
The chain rule states that if y = f(g(x)), then dy/dx =
  • A) f(g'(x)) + g(f'(x))
  • B) f'(g(x)) · g'(x)
  • C) f'(x) · g'(x)
  • D) f'(g'(x))
B is correct. f'(g(x)) · g'(x)
Q5
The Mean Value Theorem states that if f is continuous on [a, b] and differentiable on (a, b), then
  • A) f'(x) = 0 somewhere on [a, b]
  • B) there exists c in (a, b) where f'(c) = [f(b) − f(a)] / (b − a)
  • C) f has a relative maximum at some c
  • D) f(a) = f(b)
B is correct. there exists c in (a, b) where f'(c) = [f(b) − f(a)] / (b − a)
Q6
A function f has a relative minimum at x = c if f'(c) = 0 and
  • A) f(c) is the global minimum
  • B) f is increasing on both sides of c
  • C) f' changes from negative to positive at c (First Derivative Test)
  • D) f''(c) = 0
C is correct. f' changes from negative to positive at c (First Derivative Test)
Q7
The integral ∫ from a to b of f(x) dx represents
  • A) the net signed area between the curve y = f(x) and the x-axis on [a, b]
  • B) the average value of f on [a, b]
  • C) the arc length of f on [a, b]
  • D) the total area regardless of sign
A is correct. the net signed area between the curve y = f(x) and the x-axis on [a, b]
Q8
The Fundamental Theorem of Calculus Part 1 states that if F(x) = ∫ from a to x of f(t) dt, then
  • A) F'(x) = f(x) — differentiation and integration are inverse operations
  • B) F(x) = f(x) at all points
  • C) the definite integral equals the area under f
  • D) F is always increasing
A is correct. F'(x) = f(x) — differentiation and integration are inverse operations
Q9
Integration by parts uses the formula
  • A) ∫ f(g(x))g'(x) dx = F(g(x)) + C
  • B) ∫ f(x)g(x) dx = F(x)G(x) + C
  • C) ∫ u dv = u'v + uv'
  • D) ∫ u dv = uv − ∫ v du, choosing u to differentiate and dv to integrate
D is correct. ∫ u dv = uv − ∫ v du, choosing u to differentiate and dv to integrate
Q10
A power series ∑ aₙ(x − c)ⁿ converges absolutely when
  • A) the series has finitely many terms
  • B) the partial sums increase without bound
  • C) the ratio test yields |x − c| < R, where R is the radius of convergence

  • A) ∑ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ — a polynomial approximation using derivatives at a single point
  • B) ∑ f(n)/n! for integer n
  • C) the derivative of f at each integer
  • D) the Maclaurin series for any a
A is correct. ∑ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ — a polynomial approximation using derivatives at a single point
Q12

  • A) must be a straight line
  • B) is unique only if f is a constant
  • C) is tangent to the slope field at every point along its path
  • D) always passes through the origin

  • A) computing the exact antiderivative
