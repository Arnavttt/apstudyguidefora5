Q1
In Java, the difference between == and .equals() for String comparison is that == checks
  • A) == works for all object types correctly
  • B) reference equality (whether two variables point to the same object), while .equals() checks value equality (same characters)
  • C) .equals() checks reference, == checks value
  • D) both perform the same comparison
B is correct. reference equality (whether two variables point to the same object), while .equals() checks value equality (same characters)
Q2
Which of the following correctly declares and initializes an integer array of size 5 in Java?
  • A) int[] arr = new int[5];
  • B) int arr = new int[5];
  • C) int arr[5];
  • D) int[] arr = {5};
A is correct. int[] arr = new int[5];
Q3
In Java, a method declared "private" can only be accessed
  • A) by subclasses through inheritance
  • B) from any class in the program
  • C) within the same class
  • D) within the same package
C is correct. within the same class
Q4
What is the output of this code: int x = 5; System.out.println(x++);
  • A) 4
  • B) a compile error
  • C) 5 — the post-increment operator returns the original value before incrementing
  • D) 6 — x is incremented first
C is correct. 5 — the post-increment operator returns the original value before incrementing
Q5
A recursive method requires a base case to
  • A) improve performance over iteration
  • B) allow the method to call other methods
  • C) pass parameters correctly
  • D) stop the recursion and prevent infinite recursive calls (stack overflow)
D is correct. stop the recursion and prevent infinite recursive calls (stack overflow)
Q6
The ArrayList class in Java differs from a regular array in that ArrayList
  • A) is more memory-efficient than arrays
  • B) can grow and shrink dynamically and provides methods like add(), remove(), and size()
  • C) can only store integers
  • D) has fixed size like a regular array
B is correct. can grow and shrink dynamically and provides methods like add(), remove(), and size()
Q7
In Java, inheritance is implemented using the keyword
  • A) implements
  • B) inherits
  • C) override
  • D) extends (a subclass extends a superclass)
D is correct. extends (a subclass extends a superclass)
Q8
Polymorphism in Java allows
  • A) a method to have multiple names
  • B) different classes to have the same exact methods with no variation
  • C) a superclass reference to call overridden methods of a subclass at runtime (dynamic dispatch)
  • D) a class to have multiple constructors only
C is correct. a superclass reference to call overridden methods of a subclass at runtime (dynamic dispatch)
Q9
What does "encapsulation" mean in object-oriented programming?
  • A) creating multiple objects from one class
  • B) using abstract classes instead of concrete ones
  • C) Bundling data (fields) and methods together in a class, and restricting direct access to internal data using private fields and public getters/setters
  • D) inheriting methods from a parent class
C is correct. Bundling data (fields) and methods together in a class, and restricting direct access to internal data using private fields and public getters/setters
Q10
In a for loop "for (int i = 0; i < 10; i++)", the loop body executes
  • A) 9 times
  • B) indefinitely if i is never updated
  • C) 10 times (i = 0, 1, 2, ..., 9)
  • D) 11 times
C is correct. 10 times (i = 0, 1, 2, ..., 9)
Q11
A binary search requires the array to be
  • A) of even length
  • B) filled with unique values only
  • C) stored as an ArrayList
  • D) sorted (in ascending or descending order) for the algorithm to work correctly
D is correct. sorted (in ascending or descending order) for the algorithm to work correctly
Q12
The worst-case time complexity of selection sort is
  • A) O(log n)
  • B) O(n²) — always makes n(n-1)/2 comparisons regardless of input order
  • C) O(n log n)
  • D) O(n)
B is correct. O(n²) — always makes n(n-1)/2 comparisons regardless of input order
Q13
In Java, an interface differs from an abstract class in that an interface
  • A) can be instantiated directly
  • B) can only have static methods
  • C) replaces all need for inheritance
  • D) cannot have instance fields or concrete methods (before Java 8 default methods), and a class can implement multiple interfaces but only extends one class
D is correct. cannot have instance fields or concrete methods (before Java 8 default methods), and a class can implement multiple interfaces but only extends one class
Q14
What is the result of integer division 7/2 in Java?
  • A) 3.5
  • B) 4
  • C) 3.5 rounded to 4
  • D) 3 — Java truncates the decimal for integer division
D is correct. 3 — Java truncates the decimal for integer division
Q15
A NullPointerException in Java is thrown when
  • A) a variable is declared but not initialized
  • B) you try to call a method or access a field on a reference variable that holds null
  • C) an array index is out of bounds
  • D) you divide an integer by zero
B is correct. you try to call a method or access a field on a reference variable that holds null
Q16
The toString() method in Java is useful because
  • A) it returns the memory address as a String
  • B) it compares two objects for equality
  • C) it provides a String representation of an object, called automatically by System.out.println()
  • D) it converts int to String using casting
C is correct. it provides a String representation of an object, called automatically by System.out.println()
Q17
In Java, the keyword "static" means the method or field belongs to
  • A) the class itself, not to any particular instance (shared among all objects)
  • B) only the current object
  • C) subclasses that override it
  • D) private access only
A is correct. the class itself, not to any particular instance (shared among all objects)
Q18
When an array is passed to a method in Java,
  • A) the method cannot modify the array
  • B) the method receives a reference to the same array (changes inside the method affect the original)
  • C) a copy of the array is made
  • D) the array becomes read-only

Q19
