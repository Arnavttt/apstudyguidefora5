Q1
A function f is one-to-one (injective) if
  • A) each output value is produced by exactly one input — it passes the horizontal line test
  • B) each input produces exactly one output (that's the definition of any function)
  • C) f(x) > 0 for all x
  • D) f is increasing on its entire domain
A is correct. each output value is produced by exactly one input — it passes the horizontal line test
Q2
The inverse function f⁻¹(x) satisfies
  • A) f⁻¹(x) = 1/f(x) (multiplicative inverse)
  • B) f⁻¹(x) = −f(x)
  • C) f⁻¹ is always defined for any f
  • D) f(f⁻¹(x)) = x and f⁻¹(f(x)) = x — it "undoes" f
D is correct. f(f⁻¹(x)) = x and f⁻¹(f(x)) = x — it "undoes" f
Q3
An exponential function f(x) = bˣ (b > 0, b ≠ 1) is increasing when
  • A) b < 1
  • B) 0 < b < 1
  • C) x > 0 always
  • D) b > 1 (the base is greater than 1)
D is correct. b > 1 (the base is greater than 1)
Q4
The natural logarithm ln(x) is the logarithm with base
  • A) π
  • B) e ≈ 2.718... (Euler's number)
  • C) 10
  • D) 2
B is correct. e ≈ 2.718... (Euler's number)
Q5
A polynomial of degree n can have at most
  • A) n real zeros (roots) and n−1 turning points (local extrema)
  • B) n+1 zeros
  • C) infinitely many zeros
  • D) exactly n turning points
A is correct. n real zeros (roots) and n−1 turning points (local extrema)
Q6
The Rational Root Theorem states that any rational zero of a polynomial with integer coefficients must be
  • A) a fraction p/q where p divides the constant term and q divides the leading coefficient
  • B) an integer factor of the leading coefficient
  • C) a factor of the degree of the polynomial
  • D) between −1 and 1
A is correct. a fraction p/q where p divides the constant term and q divides the leading coefficient
Q7
The period of f(x) = sin(bx) is
  • A) π/b
  • B) 2π/b
  • C) 2πb
  • D) b/2π
B is correct. 2π/b
Q8
The amplitude of f(x) = A sin(x) is
  • A) A (which can be negative)
  • B) 2A
  • C) A/2
  • D) |A| — the maximum displacement from the midline
D is correct. |A| — the maximum displacement from the midline
Q9
A unit circle definition of sine and cosine gives sin(θ) and cos(θ) as
  • A) the slope and y-intercept of the terminal ray
  • B) the y-coordinate and x-coordinate of the point where the terminal ray intersects the unit circle
  • C) the ratio of adjacent/hypotenuse and opposite/hypotenuse
  • D) the arc length and radius of a circle
B is correct. the y-coordinate and x-coordinate of the point where the terminal ray intersects the unit circle
Q10
The Pythagorean identity states that for any angle θ
  • A) tan²(θ) + 1 = sec²(θ) only
  • B) sin(θ)/cos(θ) = 1
  • C) sin(θ) × cos(θ) = 1/2
  • D) sin²(θ) + cos²(θ) = 1
D is correct. sin²(θ) + cos²(θ) = 1
Q11
A geometric sequence has a common ratio r, meaning consecutive terms satisfy
  • A) aₙ = aₙ₋₁ + r (common difference)
  • B) aₙ = r × n (proportional to n)
  • C) aₙ = aₙ₋₁ − r
  • D) aₙ = aₙ₋₁ × r (each term is multiplied by a constant ratio)
D is correct. aₙ = aₙ₋₁ × r (each term is multiplied by a constant ratio)
Q12
The sum of an infinite geometric series converges when
  • A) r > 0
  • B) |r| > 1
  • C) the series has at least 10 terms
  • D) |r| < 1, and the sum is a₁/(1 − r)
D is correct. |r| < 1, and the sum is a₁/(1 − r)
Q13
To find the vertical asymptote of a rational function f(x) = p(x)/q(x), set
  • A) p(x) = 0
  • B) p(x) = q(x)
  • C) the numerator degree > denominator degree
  • D) q(x) = 0 and verify p(x) ≠ 0 at those values
D is correct. q(x) = 0 and verify p(x) ≠ 0 at those values
Q14
The horizontal asymptote of f(x) = (3x² + 1)/(x² − 4) as x → ±∞ is
  • A) y = 0 (numerator degree < denominator degree)
  • B) y = −1/4
  • C) no horizontal asymptote
  • D) y = 3 (ratio of leading coefficients when degrees are equal)
D is correct. y = 3 (ratio of leading coefficients when degrees are equal)
Q15
An even function satisfies f(−x) = f(x), meaning it is symmetric about
  • A) the origin
  • B) the x-axis
  • C) y = x
  • D) the y-axis
D is correct. the y-axis
Q16
The change of base formula for logarithms is
  • A) log_b(x) = b × log(x)
  • B) log_b(x) = ln(b)/ln(x)
  • C) log_b(x) = log(b)/log(x)
  • D) log_b(x) = ln(x)/ln(b) = log(x)/log(b)
D is correct. log_b(x) = ln(x)/ln(b) = log(x)/log(b)
Q17
Completing the square on x² + bx converts it to
  • A) (x + b)² − b²
  • B) (x + b/2)²
  • C) (x − b/2)² + (b/2)²
  • D) (x + b/2)² − (b/2)²
D is correct. (x + b/2)² − (b/2)²
Q18
A complex number z = a + bi has modulus (absolute value) equal to
  • A) √(a² + b²)
  • B) a + b
  • C) |a| + |b|
  • D) √(a² − b²)
A is correct. √(a² + b²)
Q19
The vertex form of a quadratic function is
  • A) f(x) = ax² + bx + c
  • B) f(x) = a(x + h)² + k
  • C) f(x) = a(x − h)² − k
  • D) f(x) = a(x − h)² + k, where (h, k) is the vertex
D is correct. f(x) = a(x − h)² + k, where (h, k) is the vertex
Q20
A polar coordinate (r, θ) converts to rectangular coordinates as
  • A) x = r cos θ, y = r sin θ
  • B) x = r sin θ, y = r cos θ
  • C) x = r/cos θ, y = r/sin θ
  • D) x = θ cos r, y = θ sin r
A is correct. x = r cos θ, y = r sin θ
