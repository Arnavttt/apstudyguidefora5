Q1
The real number property a(b + c) = ab + ac is called the
  • A) Associative Property
  • B) Identity Property
  • C) Distributive Property
  • D) Commutative Property
C is correct. Distributive Property
Q2
To simplify the expression (x^3)(x^5), you
  • A) subtract the exponents: x^-2
  • B) keep the larger exponent: x^5
  • C) add the exponents: x^(3+5) = x^8
  • D) multiply the exponents: x^15
C is correct. add the exponents: x^(3+5) = x^8
Q3
The solution to |x - 3| = 7 is
  • A) x = 4 or x = -4
  • B) x = 10 or x = 4
  • C) x = 10 or x = -4
  • D) x = 10 only
C is correct. x = 10 or x = -4
Q4
A linear equation in one variable has
  • A) no solutions
  • B) infinitely many solutions always
  • C) exactly one solution (unless it is an identity or contradiction)
  • D) always exactly two solutions
C is correct. exactly one solution (unless it is an identity or contradiction)
Q5
The slope of a line passing through (2, 5) and (6, 13) is
  • A) 8
  • B) 2, calculated as (13-5)/(6-2) = 8/4 = 2
  • C) 1/2
  • D) 4
B is correct. 2, calculated as (13-5)/(6-2) = 8/4 = 2
Q6
A function f is defined as a relation where
  • A) each input (x-value) maps to exactly one output (y-value)
  • B) each output maps to exactly one input
  • C) inputs and outputs can be any relation
  • D) the graph must be a straight line
A is correct. each input (x-value) maps to exactly one output (y-value)
Q7
The vertical line test is used to determine whether
  • A) a function is linear
  • B) a graph has a vertical asymptote
  • C) a graph represents a function — if any vertical line crosses the graph more than once, it is not a function
  • D) a function is one-to-one (injective)
C is correct. a graph represents a function — if any vertical line crosses the graph more than once, it is not a function
Q8
The domain of f(x) = 1/(x - 4) excludes
  • A) x = -4
  • B) all negative values of x
  • C) x = 4, because division by zero is undefined
  • D) x = 0
C is correct. x = 4, because division by zero is undefined
Q9
The vertex form of a quadratic function is
  • A) f(x) = ax^2 + bx + c (standard form)
  • B) f(x) = (x - r)(x - s) (factored form)
  • C) f(x) = mx + b (slope-intercept form)
  • D) f(x) = a(x - h)^2 + k, where (h, k) is the vertex
D is correct. f(x) = a(x - h)^2 + k, where (h, k) is the vertex
Q10
The quadratic formula gives the solutions to ax^2 + bx + c = 0 as
  • A) x = ±√(c/a)
  • B) x = [-b ± √(b^2 - 4ac)] / (2a)
  • C) x = -b/a
  • D) x = (b ± √discriminant) / a
B is correct. x = [-b ± √(b^2 - 4ac)] / (2a)
Q11
If the discriminant b^2 - 4ac is negative, the quadratic has
  • A) one repeated real solution
  • B) two distinct real solutions
  • C) no solutions of any kind
  • D) no real solutions — only two complex (imaginary) solutions
D is correct. no real solutions — only two complex (imaginary) solutions
Q12
The function f(x) = x^2 - 4 factors as
  • A) (x + 2)(x - 2) — a difference of squares
  • B) (x - 2)^2
  • C) (x + 4)(x - 1)
  • D) (x - 2)(x + 4)
A is correct. (x + 2)(x - 2) — a difference of squares
Q13
The composition (f ∘ g)(x) means
  • A) f(x) · g(x)
  • B) g(f(x)) — apply f first then g
  • C) the sum f(x) + g(x)
  • D) f(g(x)) — apply g first, then apply f to the result
D is correct. f(g(x)) — apply g first, then apply f to the result
Q14
The inverse function f^(-1) of f satisfies
  • A) f(f^(-1)(x)) = x and f^(-1)(f(x)) = x — it "undoes" f
  • B) f^(-1)(x) = 1/f(x) (the reciprocal)
  • C) the inverse always exists for every function
  • D) f^(-1)(x) = -f(x)
A is correct. f(f^(-1)(x)) = x and f^(-1)(f(x)) = x — it "undoes" f
Q15
To find the inverse of y = 2x + 3, you
  • A) divide all terms by 2: y = x + 3/2
  • B) reflect the graph over the x-axis
  • C) swap x and y, then solve: x = 2y + 3 → y = (x - 3)/2, so f^(-1)(x) = (x-3)/2
  • D) take the negative of each term: y = -2x - 3
C is correct. swap x and y, then solve: x = 2y + 3 → y = (x - 3)/2, so f^(-1)(x) = (x-3)/2
Q16
The slope of a horizontal line is
  • A) 1
  • B) infinite
  • C) 0 — the line has no rise for any run
  • D) undefined
C is correct. 0 — the line has no rise for any run
Q17
A system of two linear equations with no solution is called
  • A) consistent
  • B) dependent
  • C) orthogonal
  • D) inconsistent — the lines are parallel and never intersect
D is correct. inconsistent — the lines are parallel and never intersect
Q18
The expression √(-16) simplifies to
  • A) 4i, where i = √(-1)
  • B) 4

A is correct. 4i, where i = √(-1)
Q19
The absolute value |x| represents
  • A) x squared
  • B) the reciprocal of x
  • C) the distance of x from zero on the number line — always non-negative

Q20
A rational expression is undefined when
  • A) its numerator equals zero
