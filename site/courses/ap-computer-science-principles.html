Q1
The binary number 1011 in base 2 equals what value in base 10?
  • A) 11 (8 + 0 + 2 + 1)
  • B) 10
  • C) 9
  • D) 12
A is correct. 11 (8 + 0 + 2 + 1)
Q2
Lossless compression differs from lossy compression in that lossless
  • A) cannot be used for images or audio
  • B) perfectly reconstructs the original data; lossy permanently discards some data to achieve higher compression
  • C) reduces file size more than lossy compression
  • D) works only with text files
B is correct. perfectly reconstructs the original data; lossy permanently discards some data to achieve higher compression
Q3
A DNS (Domain Name System) server translates
  • A) human-readable domain names (e.g., google.com) into IP addresses that computers use to locate each other
  • B) IP addresses into MAC addresses
  • C) URLs into HTML code
  • D) packets into data frames
A is correct. human-readable domain names (e.g., google.com) into IP addresses that computers use to locate each other
Q4
Encryption protects data by
  • A) compressing data so it is too small to read
  • B) deleting metadata from files
  • C) storing data in multiple locations
  • D) transforming it into an unreadable form using a cipher key, so only authorized parties with the key can decrypt it
D is correct. transforming it into an unreadable form using a cipher key, so only authorized parties with the key can decrypt it
Q5
The concept of "abstraction" in programming means
  • A) using variables instead of constants
  • B) writing code in multiple programming languages
  • C) hiding complex implementation details behind a simpler interface, so users work with what a function does, not how it does it
  • D) breaking a program into smaller pieces
C is correct. hiding complex implementation details behind a simpler interface, so users work with what a function does, not how it does it
Q6
In a sequential algorithm, statements are executed
  • A) randomly by the computer
  • B) one after another in order, from top to bottom
  • C) simultaneously in parallel
  • D) only if conditions are met
B is correct. one after another in order, from top to bottom
Q7
The internet uses packet switching, which means
  • A) connections are dedicated circuits for each session
  • B) packets always arrive in order
  • C) data is broken into small packets that may travel different paths and are reassembled at the destination
  • D) all data travels the same fixed path from source to destination
C is correct. data is broken into small packets that may travel different paths and are reassembled at the destination
Q8
An algorithm is considered efficient if
  • A) it always runs in under one second
  • B) it uses no more than 100 lines of code
  • C) it works correctly for exactly three test cases
  • D) its runtime grows slowly (e.g., O(log n) or O(n)) as the input size n grows
D is correct. its runtime grows slowly (e.g., O(log n) or O(n)) as the input size n grows
Q9
Creative Commons licenses allow creators to
  • A) sell their work exclusively online
  • B) prevent all copying of digital content
  • C) copyright their work indefinitely
  • D) specify how others may use their work (e.g., with attribution, noncommercially, or with share-alike requirements)
D is correct. specify how others may use their work (e.g., with attribution, noncommercially, or with share-alike requirements)
Q10
A "bug" in a program is
  • A) a feature intentionally designed for testing
  • B) a slow but correct algorithm
  • C) a type of comment in the code
  • D) an error in the code that causes incorrect or unexpected behavior
D is correct. an error in the code that causes incorrect or unexpected behavior
Q11
The key advantage of using functions (procedures) in a program is
  • A) preventing the use of loops
  • B) code reusability and modularity — write once, call many times, reducing duplication and errors
  • C) making the program run faster in all cases
  • D) allowing the program to use more memory
B is correct. code reusability and modularity — write once, call many times, reducing duplication and errors

  • A) IPv4 could not support wireless networks
  • B) IPv4's 32-bit addresses provide only ~4 billion unique addresses, which are nearly exhausted
  • C) IPv4 was vulnerable to hacking
  • D) IPv6 is faster by design

Q13
The term "bandwidth" in networking refers to
  • A) the round-trip time for a packet
  • B) the maximum rate at which data can be transmitted over a network connection (often in Mbps or Gbps)
