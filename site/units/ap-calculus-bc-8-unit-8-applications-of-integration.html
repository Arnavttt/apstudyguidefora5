Average Value and Mean Value Theorem for Integrals

Learn on this site
Average Value Formula

f_avg=(1/(b−a))∫_a^b f(x)dx. Divide the integral by the interval length. Represents the constant height whose rectangle has the same area as under f.

Common misconception: Students use (f(a)+f(b))/2 (trapezoidal average) instead of the integral formula; only the integral gives the true average value.

How it can be tested: Compute average value using the integral formula and interpret it in context.

Mean Value Theorem for Integrals

If f is continuous on [a,b], there exists c in (a,b) such that f(c)=f_avg. Guarantees a point where f equals its average.

Common misconception: Students confuse MVT for integrals with MVT for derivatives; one guarantees f(c)=average value, the other f'(c)=average rate.

How it can be tested: Apply MVT for integrals to guarantee c and find its value for specific functions.

Interpreting Average Value in Context

Average value of a rate gives average rate; average value of a position gives average position. Always include units and interpret in the specific context given.

Common misconception: Students compute average value correctly but neglect to interpret what it means in the stated context.

How it can be tested: State average value with correct units and give a sentence interpreting its meaning.

Key Vocabulary
  • average value — f_avg=(1/(b−a))∫_a^b f dx
  • MVT for integrals — ∃c∈(a,b) with f(c)=f_avg
  • interpretation — Average value represents the constant level with equal area
  • units — Average value has same units as f(x)
Objectives
  • Compute average value using integration
  • Apply MVT for integrals to find c
  • Interpret average value in physical context
How this can be tested
  • Average value appears in both MC and FR
  • MVT for integrals tested as justify/explain
  • Always requires interpretation in FR
Practice before moving on
  • Find average value of f(x)=x² on [0,3]
  • Find c guaranteed by MVT for integrals for f(x)=cos x on [0,π/2]

Volumes of Solids of Revolution

Learn on this site
Disk Method

Volume=π∫_a^b [f(x)]² dx. Rotate a region bounded by f and the x-axis around the x-axis. Each cross-section is a disk with radius f(x).

Common misconception: Students forget the π and the square; V=π∫r² dx, not ∫r dx.

How it can be tested: Set up and evaluate disk method integrals for revolution around the x-axis.

Washer Method

Volume=π∫_a^b ([R(x)]²−[r(x)]²) dx, where R is outer radius and r is inner radius. Used when the solid has a hole.

Common misconception: Students use R²−r² inside the square: (R−r)² ≠ R²−r². The formula subtracts areas, not radii.

How it can be tested: Set up washer method when the region is between two curves rotated around an axis.

Shell Method

Volume=2π∫_a^b x·f(x)dx. Rotating a region around the y-axis using vertical strips. Useful when washer method is cumbersome.

Common misconception: Students use disk/washer for y-axis revolution; shell method avoids re-expressing x=g(y) by keeping integration in x.

How it can be tested: Set up and evaluate shell method integrals for revolution around the y-axis.

Key Vocabulary
  • disk method — V=π∫[f(x)]²dx; rotation around x-axis
  • washer method — V=π∫([R]²−[r]²)dx; rotation with hole
  • shell method — V=2π∫x·f(x)dx; rotation around y-axis
  • axis of revolution — The line around which the region is rotated
Objectives
  • Set up and evaluate disk, washer, and shell integrals
  • Identify outer and inner radius for washer problems
  • Choose the appropriate method based on axis and region
How this can be tested
  • Volume problems are guaranteed in AP® FR (Units 8)
  • Washer method most common; shell occasionally
  • Must set up integral correctly — partial credit for setup even with arithmetic errors
Practice before moving on
  • Find volume of y=√x on [0,4] rotated around x-axis (disk)
  • Find volume between y=x and y=x² rotated around x-axis (washer)

Arc Length and Surface Area (BC)

Learn on this site
Arc Length Formula

L=∫_a^b √(1+[f'(x)]²) dx. Derived by summing infinitesimal hypotenuses ds=√(dx²+dy²) along the curve.

Common misconception: Students forget the 1+ inside the square root; without it, the formula would give the projected length, not the actual curve length.

How it can be tested: Set up arc length integrals; recognize they often cannot be simplified without a calculator.

Parametric Arc Length

For x=f(t), y=g(t): L=∫_α^β √((dx/dt)²+(dy/dt)²) dt. This is the integral of the speed over the time interval.

Common misconception: Students use the Cartesian arc length formula for parametric curves; the parametric version integrates speed, not 1+(dy/dx)².

How it can be tested: Set up parametric arc length integral; identify the integrand as speed.

Surface Area of Revolution (BC)

S=2π∫_a^b f(x)√(1+[f'(x)]²)dx when rotating around x-axis. Combines arc length with the circumference at each point.

Common misconception: Students confuse surface area with volume; surface area integrates 2πr·ds (strip of surface), volume integrates πr²·dx (disk).

How it can be tested: Set up surface area integrals and distinguish them from volume integrals.

Key Vocabulary
  • arc length — L=∫√(1+(f'(x))²)dx; total length along curve
  • parametric arc length — L=∫√((dx/dt)²+(dy/dt)²)dt = ∫speed dt
  • surface area — S=2π∫f(x)√(1+(f'(x))²)dx; rotating curve around x-axis
  • ds — Infinitesimal arc length element: √(dx²+dy²)
Objectives
  • Set up arc length integrals from function formula
  • Set up parametric arc length as integral of speed
  • Distinguish surface area from volume integrals
How this can be tested
  • Arc length tested in BC FR — must know the setup
  • Parametric arc length = speed integral (important connection)
  • Surface area in BC MC and FR
Practice before moving on
  • Set up arc length of y=x² on [0,2]
  • Set up arc length of x=cos t, y=sin t for t∈[0,π]

Integration in Motion and Accumulation Contexts

Learn on this site
Position from Velocity

If v(t) is given and s(t₀) is known, then s(t)=s(t₀)+∫_{t₀}^t v(τ)dτ. Total distance = ∫|v(t)|dt; displacement = ∫v(t)dt.

Common misconception: Students use total area under v(t) for both displacement and distance; only total area gives distance when split at sign changes of v.

How it can be tested: Compute position at time t using FTC; compute total distance by splitting at zeros of v.

Accumulated Change from Rates

∫_a^b r(t)dt = total accumulated change. Works for any rate: flow rate, birth rate, production rate. Always interpret with units.

Common misconception: Students compute the integral correctly but forget to add the initial quantity; ∫r dt gives the change, not the final amount.

How it can be tested: Calculate total accumulation from a rate and add to initial condition to find final amount.

Analyzing Particle Motion with Integrals

Speed: |v(t)|. Speeding up: v·a>0. To find farthest right: integrate v from start, find when v=0, check cumulative displacement.

Common misconception: Students find where v=0 and assume that's the farthest point; they must also check whether the particle moved right or left before that zero.

How it can be tested: Determine position of a particle at specific times by integrating velocity from known initial position.

Key Vocabulary
  • displacement — ∫v(t)dt; signed, gives net position change
  • total distance — ∫|v(t)|dt; always positive, splits at zeros of v
  • accumulated change — ∫rate dt; final − initial amount
  • initial value — Must be added to the integral to get final position or amount
Objectives
  • Compute displacement and total distance from velocity
  • Add integral of rate to initial value to find final amount
  • Find position at specific time from v(t) and s(t₀)
How this can be tested
  • Motion FR problems appear every year on the AP® exam
  • Displacement vs. total distance distinguished in FR
  • Rate problems require adding initial amount to integral
