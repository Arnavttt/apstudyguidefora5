Exponential Functions

An exponential function has the form f(x) = ab^x where b>0, b≠1, a≠0. If b>1, the function grows exponentially; if 0

Common misconception: Students confuse x^2 (polynomial, grows slower eventually) with 2^x (exponential, always eventually faster); exponential functions dominate polynomial functions for large x.

How it can be tested: Identify exponential growth vs. decay; apply compound interest formulas; graph exponential functions and describe transformations.

Logarithms and Their Properties

The logarithm log_b(x) = y means b^y = x. Properties: log_b(MN) = log_b(M)+log_b(N); log_b(M/N) = log_b(M)-log_b(N); log_b(M^p) = p·log_b(M). Change of base: log_b(x) = ln(x)/ln(b). To solve exponential equations, take the logarithm of both sides; to solve logarithmic equations, exponentiate both sides (and check for domain restrictions).

Common misconception: Students think log(a+b) = log(a) + log(b) — there is no such property. Only products and quotients, not sums, split under logarithms.

How it can be tested: Apply logarithm properties to expand/condense; solve exponential and logarithmic equations; apply to real-world growth and decay problems.

Key Vocabulary
  • exponential function — A function of the form f(x) = ab^x where the variable is in the exponent
  • logarithm — The inverse of the exponential: log_b(x) = y means b^y = x
  • natural logarithm — ln(x) = log_e(x); the inverse of e^x
  • change of base formula — log_b(x) = log(x)/log(b) = ln(x)/ln(b) — converts any log to base 10 or base e
  • half-life — The time for an exponentially decaying quantity to reduce to half its initial value
  • asymptote — For exponential functions f(x) = ab^x, the x-axis (y=0) is a horizontal asymptote

  • A) 6

Q3

  • A) the chain rule of calculus
  • B) the distributive property
  • C) the exponent rule b^m · b^n = b^(m+n) — multiplication becomes addition in log form

  • A) x > 0
