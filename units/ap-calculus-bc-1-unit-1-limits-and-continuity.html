Intuitive and Formal Definition of a Limit

Learn on this site
What a Limit Means

A limit describes the value a function approaches as the input approaches a given point, not necessarily the function's value there.

Common misconception: Students confuse f(a) with lim_{x→a} f(x); limits describe approach, not the actual value.

How it can be tested: Given a graph or table, identify lim_{x→a} f(x) vs f(a).

One-Sided Limits

The left-hand limit (x→a⁻) and right-hand limit (x→a⁺) must both exist and be equal for the two-sided limit to exist.

Common misconception: A two-sided limit can fail to exist even if both one-sided limits exist but differ.

How it can be tested: Evaluate one-sided limits from piecewise graphs; determine if two-sided limit exists.

Limits at Infinity

As x→±∞, rational functions approach the ratio of leading coefficients if degrees match, 0 if denominator degree is higher, or ±∞ if numerator degree is higher.

Common misconception: Students forget to compare degrees; for equal degrees the limit is the ratio of leading coefficients, not 1.

How it can be tested: Simplify rational expressions as x→∞ using dominant-term analysis.

Key Vocabulary
  • limit — The value a function approaches as the input nears a particular point
  • one-sided limit — A limit evaluated as x approaches a value from only one direction
  • limit at infinity — The value a function approaches as x grows without bound
  • indeterminate form — An expression like 0/0 or ∞/∞ that requires further analysis
Objectives
  • Evaluate limits from graphs and tables
  • Distinguish one-sided from two-sided limits
  • Determine when limits at infinity are finite or infinite
How this can be tested
  • Multiple choice: read limit from graph or table
  • Free response: justify why a limit does or does not exist
  • Algebraic simplification before evaluating limit
Practice before moving on
  • Sketch a function where f(2)=5 but lim_{x→2}f(x)=3
  • Practice limit tables with values approaching from both sides
Key Formulas
  • Limit: \( \lim_{x \to c} f(x) = L \)
  • Formal (ε–δ): \( \forall\,\varepsilon>0\;\exists\,\delta>0 : 0<|x-c|<\delta \Rightarrow |f(x)-L|<\varepsilon \)
  • Limit at ∞: \( \lim_{x \to \infty} \frac{a_n x^n}{b_m x^m} \)

Evaluating Limits Algebraically

Learn on this site
Direct Substitution and Indeterminate Forms

If direct substitution yields 0/0 or ∞/∞, algebraic manipulation (factoring, rationalizing) is needed to resolve the indeterminate form.

Common misconception: Students stop at 0/0 and say the limit does not exist; 0/0 is indeterminate, not undefined — it requires further work.

How it can be tested: Factor and cancel common factors to evaluate limits algebraically.

Limit Laws

Limit laws allow limits of sums, products, and quotients to be computed from individual limits, provided the component limits exist.

Common misconception: The quotient law requires the denominator limit to be non-zero; if it is zero, the law cannot be applied directly.

How it can be tested: Apply sum, product, and quotient laws step by step on multi-choice problems.

Squeeze Theorem

If g(x) ≤ f(x) ≤ h(x) near a and lim g = lim h = L, then lim f = L. Used for oscillating functions like x²sin(1/x).

Common misconception: Students try to apply squeeze theorem without verifying the inequality holds near the point.

How it can be tested: Set up and apply squeeze theorem for x²sin(1/x) as x→0.

Key Vocabulary
  • direct substitution — Plugging the limit point directly into the function; valid when f is continuous there
  • factoring technique — Canceling common (x−a) factors from numerator and denominator to resolve 0/0
  • rationalizing — Multiplying by conjugate to eliminate radicals in limit expressions
  • Squeeze Theorem — If a function is bounded between two functions with the same limit, it shares that limit
Objectives
  • Factor and cancel to resolve 0/0 indeterminate forms
  • Apply limit laws to break complex limits into parts
  • Use squeeze theorem for oscillating or bounded functions
How this can be tested
  • Algebraic limit problems appear on both MC and FR sections
  • Expect to show all algebraic steps in free response
  • Squeeze theorem may appear as a justify/explain question
Practice before moving on
  • Show all steps when factoring to simplify
  • Practice conjugate multiplication for limits with radicals
Key Formulas
  • Squeeze Theorem: \( g(x)\le f(x)\le h(x),\;\lim g=\lim h=L \;\Rightarrow\; \lim f=L \)
  • Fund. trig limit: \( \lim_{x \to 0}\dfrac{\sin x}{x}=1 \)

Continuity

Learn on this site
Definition of Continuity

A function is continuous at x=a if: (1) f(a) is defined, (2) lim_{x→a}f(x) exists, and (3) lim_{x→a}f(x)=f(a). All three conditions must hold.

Common misconception: Students think a function is continuous if "the graph looks smooth"; holes, jumps, and vertical asymptotes all violate continuity.

How it can be tested: Given a piecewise function, find values of constants that make it continuous at a point.

Types of Discontinuities

Removable (hole): limit exists but ≠ f(a). Jump: one-sided limits are unequal. Infinite: vertical asymptote. Each has a distinct appearance on a graph.

Common misconception: Students call all discontinuities "holes"; removable, jump, and infinite discontinuities are categorically different.

How it can be tested: Classify a discontinuity from a graph or formula and justify the classification.

Intermediate Value Theorem

If f is continuous on [a,b] and N is between f(a) and f(b), then there exists c in (a,b) with f(c)=N. Used to guarantee existence of zeros.

Common misconception: IVT guarantees existence of c but does not find c; students confuse it with actually solving the equation.

How it can be tested: Use IVT to argue a function has a root on a given interval.

Key Vocabulary
  • continuous function — A function with no holes, jumps, or vertical asymptotes on its domain
  • removable discontinuity — A hole in the graph where the limit exists but f(a) is wrong or missing
  • jump discontinuity — A discontinuity where left- and right-hand limits exist but are unequal
  • Intermediate Value Theorem — Guarantees a continuous function takes every value between f(a) and f(b)
Objectives
  • Verify all three continuity conditions at a point
  • Classify discontinuities by type with justification
  • Apply IVT to prove a root exists on an interval
How this can be tested
  • Piecewise functions tested for continuity by setting one-sided limits equal
  • IVT appears as justify/explain free-response questions
  • Graphs tested for identifying and classifying discontinuities
Practice before moving on
  • Find k so f(x)={x²+1, x<2; kx−1, x≥2} is continuous
  • Verify or refute continuity of f(x)=1/(x−3) on [1,5]
Key Formulas
  • 3 conditions: \( f(a) \text{ exists},\quad \lim_{x\to a}f(x) \text{ exists},\quad \lim_{x\to a}f(x)=f(a) \)
  • IVT: \( f \text{ continuous on }[a,b],\;N\in(f(a),f(b)) \Rightarrow \exists\,c:f(c)=N \)

Infinite Limits and L'Hôpital's Rule (BC)

Learn on this site
Vertical Asymptotes and Infinite Limits

If lim_{x→a}f(x)=±∞, the line x=a is a vertical asymptote. This occurs when the denominator→0 and numerator→nonzero.

Common misconception: Students confuse "limit is infinity" with "limit does not exist"; technically both are true — the limit is unbounded hence DNE, but we write ±∞ to describe behavior.

How it can be tested: Determine whether f→+∞ or −∞ from either side of a vertical asymptote.

L'Hôpital's Rule

If lim f/g yields 0/0 or ∞/∞, then lim f/g = lim f'/g', provided the new limit exists. Apply repeatedly if needed.

Common misconception: Students apply L'Hôpital without verifying the indeterminate form; it is only valid for 0/0 or ∞/∞.

How it can be tested: Verify indeterminate form, then differentiate numerator and denominator separately before re-evaluating.

Improper Integrals via Limits (BC)

An improper integral ∫_a^∞ f dx is defined as lim_{t→∞}∫_a^t f dx. It converges if the limit is finite, diverges otherwise.

Common misconception: Students evaluate improper integrals as if the bound is a normal number; a limit must replace the infinite bound.

How it can be tested: Set up and evaluate improper integrals using limit notation; classify as convergent or divergent.

Key Vocabulary
  • vertical asymptote — A vertical line x=a where f(x)→±∞
  • L'Hôpital's Rule — lim f/g = lim f'/g' when the form is 0/0 or ∞/∞
  • improper integral — An integral with an infinite bound or discontinuous integrand, evaluated as a limit
  • convergent/divergent — Whether an improper integral approaches a finite value or not
Objectives
  • Identify vertical asymptotes and determine sign of infinite limit
  • Apply L'Hôpital's Rule correctly to 0/0 and ∞/∞ forms
  • Set up and evaluate basic improper integrals
How this can be tested
  • Infinite limit behavior tested via sign analysis near asymptote
  • L'Hôpital's Rule is BC only but may appear on AB as a limit technique
  • Improper integrals tested on BC free response
Practice before moving on
  • Evaluate lim_{x→0⁺} ln(x)/x using L'Hôpital
  • Determine convergence of ∫_1^∞ 1/x dx
