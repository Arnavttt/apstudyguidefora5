Parametric Equations and Calculus

Learn on this site
Area Under Parametric Curve

A=∫y dx=∫_α^β y(t)·x'(t)dt. When x increases (x'(t)>0), the formula gives the area between the curve and x-axis.

Common misconception: Students compute ∫y(t)dt instead of ∫y(t)·x'(t)dt; the dx must be replaced by x'(t)dt.

How it can be tested: Set up area under a parametric curve by replacing dx with x'(t)dt in the area integral.

Parametric Arc Length

L=∫_α^β √((dx/dt)²+(dy/dt)²) dt. Interpret as integrating speed over time.

Common misconception: Students use (dy/dx)² form instead of ((dx/dt)²+(dy/dt)²); the parametric form avoids computing dy/dx.

How it can be tested: Set up and evaluate parametric arc length by computing dx/dt and dy/dt separately.

Second Derivative (Concavity) in Parametric

d²y/dx²=[d(dy/dx)/dt]/(dx/dt). This is NOT g''(t)/f''(t). Must differentiate dy/dx with respect to t, then divide by dx/dt.

Common misconception: Students compute g''(t)/f''(t) for d²y/dx²; the correct formula differentiates dy/dx (a t-function) then divides by dx/dt.

How it can be tested: Compute second derivative of parametric curve using the correct two-step formula.

Key Vocabulary
  • parametric area — A=∫y(t)·x'(t)dt
  • parametric arc length — L=∫√((x'(t))²+(y'(t))²)dt
  • parametric d²y/dx² — [d(dy/dx)/dt]/(dx/dt)
  • speed (parametric) — |v(t)|=√((x'(t))²+(y'(t))²)
Objectives
  • Set up area under parametric curves
  • Compute arc length using speed integral
  • Find d²y/dx² correctly for parametric curves
How this can be tested
  • Parametric calculus in BC FR — all three types may appear
  • Area under parametric curve common in BC MC
  • Second derivative tested in BC FR
Practice before moving on

Polar Coordinates and Calculus

Learn on this site
Polar Coordinates Basics

Point (r,θ): r is distance from origin, θ is angle from positive x-axis. Convert: x=r cos θ, y=r sin θ, r²=x²+y².

Common misconception: Students swap sin and cos: x=r cos θ (not sin), y=r sin θ (not cos).

How it can be tested: Convert between polar and rectangular; plot polar curves like r=1+cos θ (cardioid).

Area in Polar Coordinates

A=(1/2)∫_α^β r² dθ. Area of a sector of angle dθ and radius r is (1/2)r²dθ; integrate over the angle range.

Common misconception: Students use ∫r dθ (circumference) instead of (1/2)∫r² dθ; the factor of 1/2 and r² come from the sector area formula.

How it can be tested: Set up and evaluate polar area integrals; find intersection angles carefully.

Slope of Polar Curves

dy/dx = (dy/dθ)/(dx/dθ). Since y=r sin θ and x=r cos θ, differentiate with product rule: dy/dθ=(dr/dθ)sin θ+r cos θ; dx/dθ=(dr/dθ)cos θ−r sin θ.

Common misconception: Students differentiate r with respect to x directly; must use chain rule through θ.

How it can be tested: Compute dy/dx for polar curves by differentiating x=r cos θ and y=r sin θ with product rule.

Key Vocabulary
  • polar conversion — x=r cos θ, y=r sin θ, r²=x²+y²
  • polar area — A=(1/2)∫_α^β r² dθ
  • polar slope — dy/dx=(dy/dθ)/(dx/dθ)
  • cardioid — r=a(1+cos θ); a common polar curve
Objectives
  • Convert points between polar and rectangular
  • Set up and evaluate polar area integrals
  • Find slope of polar curves using parametric differentiation
How this can be tested
  • Polar area tested in BC FR — guaranteed
  • Area between two polar curves common
  • Polar slope in BC MC
Practice before moving on

  • Find slope of r=1+sin θ at θ=π/2

Vector-Valued Functions

Learn on this site
Position, Velocity, Acceleration Vectors

r(t)=⟨x(t),y(t)⟩; v(t)=r'(t)=⟨x'(t),y'(t)⟩; a(t)=r''(t). Speed=|v|=√(x'²+y'²). Displacement=r(b)−r(a).

Common misconception: Students compute speed as |x'|+|y'| (wrong); speed is the magnitude √(x'²+y'²), not the sum of component magnitudes.

How it can be tested: Compute velocity, speed, and acceleration from a position vector; find displacement.

Total Distance vs. Displacement (Vector)

Total distance = ∫_a^b|v(t)|dt = ∫_a^b√(x'²+y'²)dt. Displacement vector = r(b)−r(a). Distance ≥ |displacement|.

Common misconception: Students compute displacement magnitude instead of integrating speed for total distance; these are different when the path curves.

How it can be tested: Set up and evaluate total distance integral for vector-valued position functions.

Integration of Vector Functions

∫r(t)dt=⟨∫x(t)dt,∫y(t)dt⟩+C. Each component integrated separately. Initial condition r(t₀)=r₀ determines the constant vector.

Common misconception: Students integrate the magnitude instead of integrating component by component.

How it can be tested: Integrate a velocity vector to find position; apply initial conditions to determine constant vector.

Key Vocabulary
  • position vector — r(t)=⟨x(t),y(t)⟩
  • velocity vector — v(t)=r'(t)=⟨x'(t),y'(t)⟩
  • speed — |v(t)|=√(x'(t)²+y'(t)²)
  • total distance (vector) — ∫|v(t)|dt = ∫√(x'²+y'²)dt
Objectives
  • Compute velocity, speed, and acceleration from position vector
  • Distinguish total distance from displacement magnitude
  • Integrate velocity vector with initial conditions
How this can be tested
  • Vector functions in BC FR — compute position from velocity
  • Speed and total distance tested
  • Integration of vector function with IC in FR

Integration in Polar and Parametric Contexts

Learn on this site
Area Between Polar Curves

A=(1/2)∫_α^β(r_outer²−r_inner²)dθ. Find intersection angles by setting r₁=r₂; split the integral at each crossing.

Common misconception: Students use r_outer−r_inner instead of r_outer²−r_inner²; the area formula squares each radius.

How it can be tested: Identify which curve is outer at each angle; integrate (r₁²−r₂²)/2 over the appropriate angle range.

Converting Polar to Parametric

Let x(θ)=r(θ)cos θ, y(θ)=r(θ)sin θ. Then calculus formulas for parametric (dy/dx, arc length, area) apply directly to polar curves.

Common misconception: Students treat polar curves as Cartesian; converting to parametric allows all parametric tools.

How it can be tested: Convert a polar equation to parametric form and apply parametric calculus formulas.

Polar Arc Length

L=∫_α^β √(r²+(dr/dθ)²)dθ. Derived from parametric arc length with x=r cos θ, y=r sin θ.

Common misconception: Students use the Cartesian arc length formula for polar curves; polar arc length uses r and dr/dθ.

How it can be tested: Set up polar arc length integral; note the formula involves r itself, not just dr/dθ.

Key Vocabulary
  • polar area between curves — A=(1/2)∫(r_outer²−r_inner²)dθ
  • polar arc length — L=∫√(r²+(dr/dθ)²)dθ
  • polar to parametric — x=r(θ)cos θ, y=r(θ)sin θ
  • intersection in polar — Solve r₁(θ)=r₂(θ) for crossing angles; also check origin
Objectives
  • Set up area between two polar curves
  • Convert polar to parametric and apply parametric formulas
  • Set up polar arc length
How this can be tested
  • Area between polar curves guaranteed in BC FR
  • Polar arc length in BC MC
  • Conversion to parametric tested as method
Practice before moving on
  • Find area inside r=2 and outside r=1+cos θ
  • Set up arc length of r=e^θ from 0 to π
