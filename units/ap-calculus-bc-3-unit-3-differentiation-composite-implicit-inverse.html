Multi-Layer Chain Rule

Learn on this site
Chaining Multiple Compositions

For y=f(g(h(x))), dy/dx=f'(g(h(x)))·g'(h(x))·h'(x). Work from outermost layer inward.

Common misconception: Students stop after one chain rule application when multiple compositions are nested.

How it can be tested: Identify all layers before differentiating multi-layer compositions.

Chain Rule with Product/Quotient Rule

When a composite function also involves a product or quotient, apply product/quotient rule first, then chain rule within each factor.

Common misconception: Students apply chain rule before product rule, mixing up the order; use product/quotient rule at the "top level" first.

How it can be tested: Differentiate expressions like (sin(x²))·(e^{3x}) requiring both product and chain rules.

Higher-Order Derivatives

f''(x) is the derivative of f'(x). Second derivatives measure concavity; f''(x)>0 means concave up. In motion, f''(t) is acceleration.

Common misconception: Students compute f''(x) by squaring f'(x) instead of differentiating f'(x) again.

How it can be tested: Find second derivatives of polynomial, trig, and exponential functions.

Key Vocabulary
  • composite function — A function formed by applying one function inside another: f(g(x))
  • higher-order derivative — The derivative of a derivative; f''(x) is the second derivative
  • concavity — f''(x)>0: concave up (cup); f''(x)<0: concave down (frown)
  • inflection point — Where concavity changes; f''(x)=0 or undefined and sign changes
Objectives
  • Apply chain rule to compositions with 3+ layers
  • Combine product/quotient and chain rules in one expression
  • Compute and interpret second derivatives
How this can be tested
  • Multi-layer chain rule tested in FR requiring explicit notation
  • Second derivatives appear in concavity and motion questions
  • Mixed rule problems common in MC

Implicit Differentiation Applications

Learn on this site
Finding Tangent Lines to Implicit Curves

After finding dy/dx implicitly, substitute the point (x,y) to get the slope, then use point-slope form. Both x and y must be substituted.

Common misconception: Students substitute only x and forget to use the y-coordinate in expressions for dy/dx.

How it can be tested: Find the equation of the tangent to x²+y²=25 at (3,4).

Second Derivative Implicitly

Differentiate dy/dx (which contains y) implicitly again with respect to x. Substitute dy/dx back to get d²y/dx² in terms of x and y only.

Common misconception: Students treat dy/dx as a constant when differentiating again; it must be differentiated as a function of x.

How it can be tested: Find d²y/dx² for y²=x implicitly.

Related Rates

Differentiate an equation relating two quantities with respect to time t. Each variable gets a rate (dV/dt, dr/dt). Given rates let you find unknown rates.

Common misconception: Students forget to differentiate implicitly with respect to t and skip the chain rule factor; every variable changing with time needs a dx/dt term.

How it can be tested: Set up related rates equations for volumes and areas and solve for an unknown rate.

Key Vocabulary
  • related rates — Differentiating an equation with respect to time to find how one rate relates to another
  • implicit curve — A curve defined by an equation in x and y that cannot easily be solved for y explicitly
  • d²y/dx² — Second derivative found by differentiating dy/dx implicitly again
  • chain rule in time — dV/dt = (dV/dr)·(dr/dt); each quantity changing with time requires this factor
Objectives
  • Find tangent lines to implicitly defined curves
  • Compute second derivatives implicitly
  • Set up and solve related rates problems
How this can be tested
  • Related rates is a guaranteed multi-part FR question on the AP® exam
  • Implicit tangent lines in MC and FR
  • Second derivative implicitly in BC FR
Practice before moving on
  • Find tangent to x²+xy+y²=7 at (1,2)
  • A ladder slides; relate dh/dt to dx/dt using implicit differentiation

Inverse Function Derivatives

Learn on this site
Inverse Function Derivative Theorem

If g=f⁻¹, then g'(x)=1/f'(g(x)). To find g'(a), find g(a) first, then evaluate 1/f'(g(a)).

Common misconception: Students try to find the inverse function formula explicitly; the theorem avoids this by relating g'sto f'.

How it can be tested: Use a table of f values to find (f⁻¹)'(a) without finding f⁻¹ explicitly.

Derivatives of arcsin and arccos

d/dx[arcsin x]=1/√(1−x²) and d/dx[arccos x]=−1/√(1−x²). These are derived from the inverse function theorem applied to sin and cos.

Common misconception: Students get the sign wrong for arccos; it is the negative of arcsin's derivative.

How it can be tested: Differentiate expressions involving arcsin x and arccos x with chain rule.

Derivatives of arctan and arccot

d/dx[arctan x]=1/(1+x²) and d/dx[arccot x]=−1/(1+x²). arctan is the most commonly tested inverse trig derivative.

Common misconception: Students forget the denominator is 1+x², not √(1−x²); arctan uses 1+x², arcsin uses √(1−x²).

How it can be tested: Differentiate arctan(x) in combination with other functions and using chain rule.

Key Vocabulary
  • arcsin — Inverse of sin; d/dx[arcsin x]=1/√(1−x²)
  • arctan — Inverse of tan; d/dx[arctan x]=1/(1+x²)
  • inverse function theorem — g'(x)=1/f'(g(x)) where g=f⁻¹
  • arccos — Inverse of cos; d/dx[arccos x]=−1/√(1−x²)
Objectives
  • Apply inverse function derivative theorem from a table
  • Differentiate arcsin and arctan with chain rule
  • Recognize when inverse trig derivatives appear in integrals
How this can be tested
  • Inverse function theorem tested via tables: "Given f(3)=7, f'(3)=2, find (f⁻¹)'(7)"
  • Arctan derivative integral forms tested in BC
  • Chain rule with inverse trig in FR
Practice before moving on
  • From a table, find (f⁻¹)'(5) given f and f' values
  • Differentiate y=arctan(x²) and y=arcsin(2x)

Mean Value Theorem and Its Applications

Learn on this site
Mean Value Theorem

If f is continuous on [a,b] and differentiable on (a,b), then ∃c where f'(c)=[f(b)−f(a)]/(b−a). The instantaneous rate equals the average rate at some point.

Common misconception: Students confuse MVT with IVT; MVT is about derivatives equaling average rate of change, not just function values.

How it can be tested: Verify MVT conditions and find c satisfying f'(c)=average rate on a given interval.

Rolle's Theorem

If f is continuous on [a,b], differentiable on (a,b), and f(a)=f(b), then ∃c where f'(c)=0. A special case of MVT.

Common misconception: Students apply Rolle's without verifying f(a)=f(b); all three conditions must hold.

How it can be tested: Verify Rolle's conditions and conclude a horizontal tangent exists.

Extreme Value Theorem

If f is continuous on a closed interval [a,b], it attains both an absolute maximum and minimum. Candidates: critical points and endpoints.

Common misconception: Students check only critical points and forget endpoints when finding absolute extrema on a closed interval.

How it can be tested: Find absolute extrema by evaluating f at all critical points and both endpoints.

Key Vocabulary
  • Mean Value Theorem — Guarantees f'(c)=average rate of change on (a,b) for some c
  • Rolle's Theorem — If f(a)=f(b), there exists c with f'(c)=0
  • Extreme Value Theorem — Continuous function on closed interval attains absolute max and min
  • critical point — Where f'(x)=0 or f'(x) is undefined; candidate for local/absolute extrema
Objectives
  • State MVT conditions and apply it to specific functions
  • Use Rolle's Theorem to guarantee a horizontal tangent
  • Find absolute extrema on a closed interval using EVT
How this can be tested
  • MVT appears as justify/explain questions in FR
  • EVT and closed interval method tested in multi-part FR
  • Rolle's Theorem in MC and FR as a theoretical guarantee
Practice before moving on
  • Verify MVT for f(x)=x²−3 on [1,4] and find c
  • Find absolute max/min of f(x)=x³−3x on [−2,2]
