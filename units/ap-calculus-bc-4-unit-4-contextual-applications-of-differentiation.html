Increasing, Decreasing, and First Derivative Test

Learn on this site
Monotonicity from f'

f'(x)>0 means f is increasing; f'(x)<0 means f is decreasing. Critical points partition the number line into sign intervals.

Common misconception: Students evaluate f at the critical point to determine behavior; they must check the sign of f' in intervals around the critical point.

How it can be tested: Create a sign chart for f'(x) and state where f increases and decreases.

First Derivative Test

If f'(x) changes from + to − at c, f has a local max at c. If f'(x) changes from − to + at c, f has a local min. No sign change means no local extremum.

Common misconception: Students classify a critical point as a max just because f''(c)=0; they must check the sign change of f' around c.

How it can be tested: Apply first derivative test to classify critical points from a graph or formula of f'.

Second Derivative Test

If f'(c)=0 and f''(c)<0, local max; if f''(c)>0, local min; if f''(c)=0, test is inconclusive — use first derivative test instead.

Common misconception: Students rely on the second derivative test when f''(c)=0, forgetting it is inconclusive in that case.

How it can be tested: Apply second derivative test where valid and switch to first derivative test when f''(c)=0.

Key Vocabulary
  • critical point — x where f'(x)=0 or undefined; potential extremum
  • first derivative test — Classify extrema by the sign change of f' around the critical point
  • second derivative test — f'(c)=0 and f''(c)<0 → local max; f''(c)>0 → local min
  • sign chart — A number line showing the sign of f' in each interval between critical points
Objectives
  • Build sign charts to find intervals of increase/decrease
  • Apply first derivative test to classify extrema
  • Use second derivative test as an alternative classification method
How this can be tested
  • f' graphs are given; students must determine behavior of f
  • FR: "On what intervals is f increasing? Justify."
  • Both derivative tests may be tested on the same exam
Practice before moving on
  • Find local extrema of f(x)=2x³−9x²+12x−4
  • Given a graph of f', identify local max/min of f

Concavity and Inflection Points

Learn on this site
Concavity from f''

f''(x)>0 on an interval means f is concave up (opens upward); f''(x)<0 means concave down. The second derivative measures how the slope is changing.

Common misconception: Students think concave up means increasing; a function can be decreasing and concave up simultaneously.

How it can be tested: Sketch a function that is decreasing and concave up; identify the role of f''.

Inflection Points

An inflection point occurs where concavity changes. f''(c) must change sign at c; f''(c)=0 alone is not sufficient (e.g., f(x)=x⁴ at x=0).

Common misconception: Students mark every zero of f'' as an inflection point; only those where f'' changes sign count.

How it can be tested: Build a sign chart for f'' and identify inflection points by sign changes.

Curve Sketching

Use f, f', and f'' together: x/y-intercepts, critical points, extrema, inflection points, and asymptotes. The complete picture requires all tools.

Common misconception: Students skip checking concavity or only use one derivative; AP® curve sketching requires explicitly justifying each feature.

How it can be tested: Produce a complete curve sketch with labeled intercepts, extrema, and inflection points.

Key Vocabulary
  • concave up — f''(x)>0; the slope of f is increasing; graph curves upward
  • concave down — f''(x)<0; the slope of f is decreasing; graph curves downward
  • inflection point — Where concavity changes; f'' changes sign
  • curve sketching — Using f, f', f'' to produce an accurate graph with all features labeled
Objectives
  • Determine concavity intervals using f''
  • Identify inflection points where f'' changes sign
  • Integrate f, f', f'' to produce a complete curve sketch
How this can be tested
  • Concavity tested from graphs of f or f'
  • Inflection points appear in both MC and FR
  • Curve sketching is a classic FR question
Practice before moving on
  • Find inflection points of f(x)=x⁴−4x³
  • Sketch f(x)=xe^{−x} using all derivative tools

Optimization

Learn on this site
Setting Up Optimization Problems

Identify the quantity to maximize/minimize (objective function) and any constraints. Express the objective in one variable using the constraint, then differentiate.

Common misconception: Students try to optimize multi-variable functions; the constraint must reduce the problem to one variable.

How it can be tested: Write the objective function in one variable using a geometric or physical constraint.

Closed Interval vs. Unbounded Domains

On a closed interval, check critical points and endpoints. On an unbounded domain, verify the single critical point is a max or min using derivative tests.

Common misconception: Students skip endpoints on closed interval problems or fail to verify that the single critical point is the desired extremum on unbounded domains.

How it can be tested: Distinguish closed-interval method from single-critical-point analysis for optimization problems.

Common Optimization Shapes

Fencing problems (rectangle with one side fixed), box problems (maximize volume with fixed surface area), distance problems (minimize distance from point to curve). Know the formulas.

Common misconception: Students set up the wrong objective function or forget a constraint (e.g., total fencing = fixed perimeter).

How it can be tested: Recognize the setup pattern for each classic problem type and derive the correct one-variable function.

Key Vocabulary
  • objective function — The quantity to be maximized or minimized
  • constraint — An equation relating the variables that limits the feasible solutions
  • closed interval method — Evaluate f at critical points and endpoints to find absolute extrema
  • optimization — Finding the max or min value of a function using calculus
Objectives
  • Translate word problems into objective + constraint equations
  • Differentiate, solve f'=0, and verify the extremum
  • Apply closed interval method or single-critical-point test appropriately
How this can be tested
  • Optimization is a guaranteed FR topic on both AB and BC
  • Must explicitly justify that the critical point is a max or min
  • Real-world context (fencing, volume, profit) always used
Practice before moving on
  • Maximize area of rectangle with 100 m of fencing along a river
  • Minimize total surface area of a cylindrical can of volume 500 cm³

Motion Along a Line

Learn on this site
Position, Velocity, and Acceleration

If s(t) is position, v(t)=s'(t) is velocity, and a(t)=v'(t)=s''(t) is acceleration. Speed = |v(t)|.

Common misconception: Students confuse speed (always non-negative) with velocity (signed). Speed = |v(t)|, not v(t).

How it can be tested: Given s(t), find v(t) and a(t); determine when the object moves right, left, or is at rest.

Speeding Up and Slowing Down

An object speeds up when v(t) and a(t) have the same sign; slows down when they have opposite signs.

Common misconception: Students think positive acceleration means speeding up; if v(t)<0 and a(t)>0, the object is actually slowing down.

How it can be tested: Determine intervals of speeding up/slowing down by comparing signs of v and a.

Total Distance vs. Displacement

Displacement = s(b)−s(a). Total distance = ∫|v(t)| dt, which requires identifying zeros of v(t) and integrating each segment separately.

Common misconception: Students use displacement (net change) when total distance is asked; they must integrate |v| or split integral at direction changes.

How it can be tested: Calculate total distance by splitting the integral at each zero of v(t).

Key Vocabulary
  • velocity — v(t)=s'(t); signed rate of change of position
  • acceleration — a(t)=v'(t); rate of change of velocity
  • speed — |v(t)|; always non-negative
  • displacement — s(b)−s(a); net change in position (signed)
Objectives
  • Compute velocity and acceleration from a position function
  • Determine when an object is speeding up or slowing down
  • Distinguish total distance from displacement
How this can be tested
  • Motion problems appear in both FR and MC
  • "When is the object at rest?" and "when is it speeding up?" are common
  • Displacement vs. total distance often distinguished in the same FR
Practice before moving on
  • s(t)=t³−6t²+9t; find when at rest, moving right, speeding up
  • Compute displacement and total distance on [0,4]
