Connecting f, f', and f''

Learn on this site
Reading f from a Graph of f'

Where f'>0, f increases; where f'<0, f decreases; where f'=0, f has a critical point. The shape of f' tells you the shape of f.

Common misconception: Students read the graph of f' as if it were f; they must translate derivative behavior back to the original function.

How it can be tested: Given a graph of f', sketch a plausible graph of f and identify all key features.

Reading f from a Graph of f''

Where f''(x)>0 (above x-axis), f is concave up; where f''<0, concave down. Where f'' changes sign, f has an inflection point.

Common misconception: Students confuse where f'' is positive (f concave up) with where f' is positive (f increasing); these are independent.

How it can be tested: From a graph of f'', describe concavity and locate inflection points of f.

All Three Graphs Together

When three separate graphs of f, f', and f'' are shown, features connect: local extrema of f appear as zeros of f'; inflection points of f appear as local extrema of f'.

Common misconception: Students treat each graph independently; the connections across all three graphs are what AP® questions test.

How it can be tested: Match features across f, f', and f'' graphs: critical points, inflection points, zeros.

Key Vocabulary
  • graph of f' — A graph showing the derivative; used to analyze original function's behavior
  • critical point on f' graph — A zero crossing of f' corresponds to a potential extremum of f
  • sign of f'' — Positive f'' graph → concave up in f; negative → concave down
  • corresponding features — Inflection points of f appear as local extrema of f'
Objectives
  • Interpret a graph of f' to determine behavior of f
  • Translate f'' graph into concavity information for f
  • Connect all three function graphs to each other
How this can be tested
  • Graph reading is a primary AP® skill; at least 2 MC questions guaranteed
  • FR parts often ask to justify using "graph of f'" language
  • Matching graphs tested in MC
Practice before moving on
  • Given graph of f', find where f has local max/min
  • Sketch f from a given f' graph

Linearization and Differentials

Learn on this site
Linear Approximation

L(x)=f(a)+f'(a)(x−a) is the tangent line approximation. Near x=a, f(x)≈L(x). Accuracy decreases as x moves away from a.

Common misconception: Students confuse the linearization formula with the tangent line equation; they are the same equation written differently.

How it can be tested: Use linearization to estimate f(a+0.1) or similar values without a calculator.

Differentials

dy=f'(x)dx. A differential approximates the change in y for a small change in x. Δy ≈ dy when Δx is small.

Common misconception: Students confuse Δy (actual change) with dy (approximate change); dy is always the tangent line approximation.

How it can be tested: Compute dy for a function and use it to estimate changes.

Error Estimation

If a quantity is computed from a measured value with error ε, the propagated error is approximately |f'(x)|·ε. Used in problems about measurement uncertainty.

Common misconception: Students calculate the error as a percentage of the function value rather than using the differential.

How it can be tested: Use differentials to estimate propagated error in calculations.

Key Vocabulary
  • linearization — L(x)=f(a)+f'(a)(x−a); tangent line used as an approximation
  • differential — dy=f'(x)dx; approximates the change in y
  • Δy vs dy — Δy is actual change; dy=f'(x)dx is the linear approximation of that change
  • error propagation — Using differentials to estimate error: Δy≈(dy/dx)·Δx
Objectives
  • Write and use linearization formula to approximate function values
  • Compute differentials and interpret them
  • Use differentials for error estimation in applied problems
How this can be tested
  • Linearization appears in BC FR and MC
  • Differentials tested in context of approximation problems
  • Often combined with: "Is this an overestimate or underestimate?"
Practice before moving on
  • Use L(x) to estimate √9.1 using f(x)=√x, a=9
  • Find dy for y=x³ and estimate Δy when x changes from 2 to 2.01

L'Hôpital's Rule in Depth

Learn on this site
All Indeterminate Forms

Beyond 0/0 and ∞/∞: 0·∞ (rewrite as 0/(1/∞)), ∞−∞ (combine fractions), 1^∞, 0⁰, ∞⁰ (take ln). Each can be converted to 0/0 or ∞/∞.

Common misconception: Students only recognize 0/0 and ∞/∞ as indeterminate; the other forms (especially 1^∞) require logarithmic conversion.

How it can be tested: Identify the indeterminate form, convert to 0/0 or ∞/∞, then apply L'Hôpital's Rule.

Repeated Application

If L'Hôpital's Rule yields another indeterminate form, apply it again. Differentiate numerator and denominator independently each time.

Common misconception: Students differentiate f/g as a quotient (quotient rule) instead of differentiating f and g separately.

How it can be tested: Apply L'Hôpital's Rule multiple times until the limit is determinate.

When L'Hôpital's Does Not Apply

If the form is not indeterminate (e.g., 2/0), L'Hôpital's Rule cannot be used; the limit is ±∞. Applying it incorrectly to non-indeterminate forms gives wrong answers.

Common misconception: Students apply L'Hôpital's Rule to 2/0, getting a wrong finite answer; verify indeterminate form first.

How it can be tested: Check the form before applying L'Hôpital's; if not indeterminate, analyze as an infinite limit.

Key Vocabulary
  • indeterminate form — 0/0, ∞/∞, 0·∞, ∞−∞, 1^∞, 0⁰, ∞⁰ — forms that require further analysis
  • L'Hôpital's Rule — If form is 0/0 or ∞/∞: lim f/g = lim f'/g'
  • 0·∞ conversion — Rewrite f·g = f/(1/g) or g/(1/f) to get 0/0 or ∞/∞
  • 1^∞ conversion — Take ln: ln(f^g)=g·ln f, then exponentiate the final limit
Objectives
  • Classify all seven indeterminate forms
  • Convert 0·∞, ∞−∞, and exponential forms to apply L'Hôpital's
  • Know when L'Hôpital's does not apply
How this can be tested
  • All seven forms can appear on BC exam
  • Exponential indeterminate forms (1^∞) common in BC MC
  • AB: primarily 0/0 and ∞/∞
Practice before moving on
  • Evaluate lim_{x→0⁺} x·ln(x) and lim_{x→∞}(1+1/x)^x
  • Evaluate lim_{x→0} (1/sin x − 1/x)

Parametric and Vector-Valued Functions (BC)

Learn on this site
Parametric Derivatives

For x=f(t), y=g(t): dy/dx=(dy/dt)/(dx/dt)=g'(t)/f'(t). Second derivative: d²y/dx²=[d(dy/dx)/dt]/(dx/dt).

Common misconception: Students compute dy/dx as g'(t)/f'(t) correctly but then compute d²y/dx² as g''(t)/f''(t) — wrong; use the formula above.

How it can be tested: Compute first and second derivatives of parametric curves.

Slope and Tangent Lines in Parametric Form

Horizontal tangents where dy/dt=0 and dx/dt≠0; vertical tangents where dx/dt=0 and dy/dt≠0. The tangent line uses the parametric slope.

Common misconception: Students find where dy/dt=0 without checking dx/dt; if both are zero at the same t, the situation is indeterminate.

How it can be tested: Find horizontal and vertical tangents of parametric curves by setting appropriate derivatives to zero.

Vectors and Motion (BC)

A particle's position vector is ⟨x(t),y(t)⟩. Velocity vector: ⟨x'(t),y'(t)⟩. Speed = √(x'²+y'²). Acceleration vector: ⟨x''(t),y''(t)⟩.

Common misconception: Students confuse speed (scalar magnitude) with velocity (vector); speed = |v| = √(x'²+y'²).

How it can be tested: Compute speed and acceleration from parametric position; find direction of motion.

Key Vocabulary
  • parametric equations — x=f(t), y=g(t); position described as function of parameter t
  • dy/dx parametric — g'(t)/f'(t) = (dy/dt)/(dx/dt)
  • speed (parametric) — |v|=√((dx/dt)²+(dy/dt)²)
  • velocity vector — ⟨dx/dt, dy/dt⟩; direction and rate of motion
Objectives
  • Compute parametric derivatives and slopes
  • Find horizontal and vertical tangents for parametric curves
  • Calculate speed and acceleration vectors
How this can be tested
  • Parametric and vector functions are BC-only; tested in FR part
  • Speed calculation commonly tested
  • Second derivative of parametric curve in BC FR
Practice before moving on
  • Find dy/dx and d²y/dx² for x=t², y=t³−t at t=1
  • Find horizontal tangents for x=sin t, y=cos(2t)
