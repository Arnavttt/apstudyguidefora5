Slope Fields

Learn on this site
Constructing Slope Fields

A slope field plots dy/dx at many points (x,y) as short line segments. Each segment has slope equal to f(x,y) evaluated at that point.

Common misconception: Students draw horizontal segments everywhere; the slope varies with both x and y in most differential equations.

How it can be tested: Plot slope field segments at given lattice points and identify which solution curve matches.

Matching Slope Fields to DEs

Match a slope field to its DE by checking: where are slopes zero? Where are they steep? Slopes that depend only on x give vertical-strip patterns; only on y give horizontal-strip patterns.

Common misconception: Students match by guessing; systematically check where dy/dx=0 to identify the zero-slope curves and eliminate wrong DEs.

How it can be tested: Identify which DE matches a given slope field by checking zero-slope isoclines.

Sketching Solution Curves

A solution curve follows the slope field — at each point, the curve's tangent matches the segment. Starting from an initial point, trace the curve through the field.

Common misconception: Students draw solution curves that cross slope field segments at wrong angles; the curve must be tangent to each segment it passes through.

How it can be tested: Given a slope field and initial condition, sketch the particular solution through that point.

Key Vocabulary
  • slope field — A diagram of short line segments showing dy/dx at many points (x,y)
  • isocline — Curve along which dy/dx is constant; e.g., dy/dx=0 is the zero-isocline
  • particular solution — The specific solution curve passing through a given initial point
  • general solution — Family of all solution curves to a DE, containing constant C
Objectives
  • Draw slope field segments at specified points
  • Match a slope field to its differential equation
  • Sketch particular solution curves through initial points
How this can be tested
  • Slope fields appear in AP® MC and FR; often the entire Unit 7 FR
  • Matching slope field to DE is a common MC question
  • Sketching a solution curve through given initial condition
Practice before moving on
  • Draw slope field for dy/dx=x−y at 9 lattice points
  • Given slope field, sketch solution with y(0)=1
Key Formulas
  • Slope field: \( \dfrac{dy}{dx}=F(x,y)\;\text{gives slope at each }(x,y) \)

Separable Differential Equations

Learn on this site
Separation of Variables

Rewrite dy/dx=f(x)g(y) as (1/g(y))dy=f(x)dx, then integrate both sides. Add +C on one side and solve for y.

Common misconception: Students add +C to both sides (only needed on one) or forget to include it at all.

How it can be tested: Separate and integrate to find the general solution; apply initial condition to find particular solution.

Applying Initial Conditions

After finding y=F(x)+C, substitute the initial condition (x₀,y₀) to solve for C. The result is the particular solution.

Common misconception: Students forget to solve for C after integrating; the general solution with undetermined C is not the particular solution.

How it can be tested: Correctly apply y(x₀)=y₀ to determine C in the general solution.

Exponential Growth and Decay

dy/dt=ky has solution y=y₀e^{kt}. If k>0: growth; k<0: decay. Half-life/doubling time derived from this model.

Common misconception: Students write y=Ce^{kt} but forget that C=y₀ (initial value) from the initial condition.

How it can be tested: Set up and solve exponential growth/decay problems from initial conditions and given rate constant.

Key Vocabulary
  • separable DE — One that can be written as g(y)dy=f(x)dx
  • general solution — y=F(x)+C; family of all solutions
  • particular solution — Specific solution after C is determined by initial condition
  • exponential model — dy/dt=ky → y=y₀e^{kt}
Objectives
  • Solve separable differential equations analytically
  • Apply initial conditions to find particular solutions
  • Set up and interpret exponential growth/decay models
How this can be tested
  • Separable DE is the #1 tested DE type on the AP® exam
  • Often appears as a multi-part FR: slope field → solve → apply IC → interpret
  • Exponential growth/decay in both context problems and pure math
Practice before moving on
  • Solve dy/dx=xy with y(0)=3
  • A bacteria population doubles in 5 hours starting at 200; write the model
Key Formulas
  • Separable DE: \( \dfrac{dy}{dx}=f(x)g(y)\;\Rightarrow\;\dfrac{dy}{g(y)}=f(x)\,dx \)
  • Exp. growth/decay: \( \dfrac{dy}{dt}=ky\;\Rightarrow\;y=Ce^{kt} \)

Euler's Method (BC)

Learn on this site
Euler's Method Algorithm

Start at (x₀,y₀). At each step: y_{n+1}=y_n+f(x_n,y_n)·Δx and x_{n+1}=x_n+Δx. Approximate the solution curve by following slope field segments.

Common misconception: Students use the new x-value when computing the slope for the current step; always use the current point (x_n, y_n) for the slope.

How it can be tested: Apply Euler's method with given step size for 2–3 steps and compare to exact solution.

Accuracy of Euler's Method

Smaller step size Δx gives a more accurate approximation. If f is concave up, Euler's underestimates; concave down, it overestimates.

Common misconception: Students think Euler's method is always an underestimate; concavity of the solution determines the direction of error.

How it can be tested: Determine whether Euler's estimate is an over- or under-estimate based on the concavity of the actual solution.

Logistic Growth (BC)

dP/dt=kP(1−P/M) where M is carrying capacity. Solution S-curves approach M asymptotically. Fastest growth at P=M/2.

Common misconception: Students confuse the logistic with exponential models; logistic growth slows as P approaches M, while exponential grows without bound.

How it can be tested: Interpret parameters in logistic model; find where growth rate is maximum.

Key Vocabulary
  • Euler's method — y_{n+1}=y_n+f(x_n,y_n)·Δx; step-by-step approximation
  • step size — Δx; smaller step gives better approximation
  • logistic growth — dP/dt=kP(1−P/M); S-shaped solution curve
  • carrying capacity — M; the upper limit that the logistic solution approaches
Objectives
  • Execute Euler's method for 2–3 steps from a given initial condition
  • Determine over/under-estimate based on concavity
  • Interpret logistic growth model parameters
How this can be tested
  • Euler's method is BC-only; tested in FR
  • Logistic growth tested in BC MC and FR
  • Over/under-estimate question common in Euler's method FR
Practice before moving on
  • Apply Euler's to dy/dx=x+y, y(0)=1, Δx=0.5, two steps
  • Describe logistic growth behavior near P=0 and P=M

Applications of Differential Equations

Learn on this site
Newton's Law of Cooling

dT/dt=k(T−T_env). Solution: T(t)=T_env+(T₀−T_env)e^{kt}, k<0. Temperature approaches ambient exponentially.

Common misconception: Students forget to subtract T_env before applying exponential formula; the DE is for (T−T_env), not T itself.

How it can be tested: Set up and solve Newton's cooling problems with given initial and ambient temperatures.

Mixing Problems

Rate of change of substance = rate in − rate out. Set up a DE for the amount Q(t) of substance in a tank, solve separably or recognize linear first-order DE.

Common misconception: Students forget that rate out depends on current concentration Q(t)/V(t), making the DE depend on Q.

How it can be tested: Write the differential equation for Q(t) in a mixing problem and identify the variables.

Interpreting Solutions in Context

After solving a DE, interpret constants and behavior: What does C represent? When does the solution approach a limit? What are the units?

Common misconception: Students report mathematical answers without contextual interpretation; AP® FR always requires context-based explanation of results.

How it can be tested: Translate mathematical results from DE solutions into meaningful contextual statements with correct units.

Key Vocabulary
  • Newton's Law of Cooling — dT/dt=k(T−T_a); T approaches ambient T_a exponentially
  • mixing problem — dQ/dt = (rate in) − (rate out); Q is amount of substance
  • initial condition — The value Q(t₀)=Q₀ used to find the particular solution
  • long-term behavior — As t→∞, solutions to these DEs approach equilibrium values
Objectives
  • Set up Newton's cooling and mixing DEs from context
  • Solve the DE with initial conditions
  • Interpret the solution in context with correct units
How this can be tested
  • Context-heavy DE problems appear in FR
  • Newton's cooling is a standard template problem
  • Interpreting long-term behavior tested in MC and FR
Practice before moving on
  • Coffee at 95°C cools in a 20°C room; write and solve the DE
  • Tank with 50L pure water; salt enters at 2g/min, exits at 2L/min
Key Formulas
  • Logistic DE: \( \dfrac{dP}{dt}=kP\!\left(1-\dfrac{P}{M}\right) \)
  • Logistic solution: \( P(t)=\dfrac{M}{1+Ae^{-kt}},\quad A=\dfrac{M-P_0}{P_0} \)
