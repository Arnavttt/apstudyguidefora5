Note Duration Relationships

Each note value is half the duration of the one above: whole = 2 halves = 4 quarters = 8 eighths = 16 sixteenths. A dot adds half the note's own value.

Common misconception: Students forget that a dotted half note = 3 beats (not 2.5) in 4/4 time; a dot is always + half of the original value.

How it can be tested: Write correct rhythmic notation for a given beat count; identify note values in a score.

Simple vs. Compound Meter

In simple meter, the beat divides into 2 (e.g., 4/4, 3/4, 2/4). In compound meter, the beat divides into 3 (e.g., 6/8, 9/8, 12/8). The top number of a compound meter gives the number of divisions, not beats.

Common misconception: Students read 6/8 as "six beats per measure"; it actually has two dotted-quarter beats, each divided into three eighth notes.

How it can be tested: Identify simple or compound meter from a time signature; determine how many beats are in a measure.

Key Vocabulary
  • simple meter — Meter where the beat divides into 2 equal parts (top numbers: 2, 3, or 4)
  • compound meter — Meter where the beat divides into 3 equal parts (top numbers: 6, 9, or 12)
  • syncopation — Accenting a normally weak beat or the off-beat, creating rhythmic tension
  • tie — A curved line connecting two notes of the same pitch, combining their durations
Objectives
  • Determine the beat value and beat count in any meter
  • Write rhythms using notes and rests
  • Distinguish simple from compound meter
How this can be tested
  • Clap and count a 6/8 excerpt
  • Rewrite a simple meter rhythm in compound meter
  • Write a measure of syncopated 4/4
Practice before moving on
  • Transcribe a 4-measure rhythm by ear using eighth note level precision

  • A) 3 quarter-note beats
