Magnetic Force on Charges and Currents

Learn on this site
Magnetic Force on a Charge

F = qv × B; magnitude F = qvB sinθ. Magnetic force is perpendicular to both v and B — it does no work on the charge.

Common misconception: Magnetic force does zero work (F perpendicular to v) — it changes direction but not speed.

How it can be tested: Calculate magnetic force on moving charges using F = qvB sinθ.

Cyclotron Motion

A charge in a uniform B perpendicular to v moves in a circle: r = mv/(qB); T = 2πm/(qB). Period is independent of speed.

Common misconception: The cyclotron radius depends on speed, but the period does NOT — all speeds have the same period in a given B.

How it can be tested: Find cyclotron radius and period for a charged particle in a magnetic field.

Magnetic Force on a Current

F = IL × B; for a straight wire of length L carrying current I in field B: F = ILB sinθ. This is the basis of electric motors.

Common misconception: Force on wire is F = ILB sinθ, not F = IB — length and angle matter.

How it can be tested: Calculate force on a current-carrying wire in a magnetic field.

Key Vocabulary
  • Magnetic force — F = qv × B; perpendicular to v and B
  • Cyclotron radius — r = mv/(qB); radius of circular path in uniform B
  • Cyclotron period — T = 2πm/(qB); independent of particle speed
  • Right-hand rule — Direction of F = qv × B; fingers from v to B, thumb gives F direction for +q
  • Force on wire — F = IL × B = ILB sinθ for straight wire
Objectives
  • Calculate F on a charge using F = qvB sinθ.
  • Find cyclotron radius and period for a given charge and field.
  • Calculate force on a current-carrying wire in a B field.
How this can be tested
  • Free-response: find path of a charge entering B field region.
  • Derivation: show that cyclotron period is independent of v.
  • Calculate force on a rectangular current loop in a uniform B.
Practice before moving on
  • Practice right-hand rule for F = qv × B.
  • Solve cyclotron radius problems for electrons and protons.

Biot-Savart Law

Learn on this site
Biot-Savart Law

dB = (μ₀/4π) I dl × r̂/r²; integrate over current distribution. μ₀ = 4π×10⁻⁷ T·m/A. For a long straight wire: B = μ₀I/(2πr).

Common misconception: The Biot-Savart law gives dB from a current element — you must integrate over the whole wire.

How it can be tested: Apply Biot-Savart to find B for straight wires, loops, and arcs.

Field of a Circular Current Loop

On the axis: B = μ₀IR²/(2(R²+x²)^{3/2}). At center (x = 0): B = μ₀I/(2R). The field is along the axis.

Common misconception: The field of a current loop on its axis decreases as 1/r³ far from the loop (like a magnetic dipole).

How it can be tested: Derive B on the axis of a current loop using Biot-Savart.

Force Between Parallel Wires

Two parallel wires with currents I₁ and I₂ separated by distance d: F/L = μ₀I₁I₂/(2πd). Parallel currents attract; antiparallel currents repel.

Common misconception: Parallel currents attract each other (not repel) — this is opposite to parallel electric charges.

How it can be tested: Calculate force per unit length between parallel current-carrying wires.

Key Vocabulary
  • μ₀ — Permeability of free space = 4π×10⁻⁷ T·m/A
  • Biot-Savart law — dB = (μ₀/4π) I dl × r̂/r²; integrate for total B
  • Long wire B — B = μ₀I/(2πr); field circles the wire; decreases as 1/r
  • Loop center B — B = μ₀I/(2R) at center of circular loop of radius R
  • Force between wires — F/L = μ₀I₁I₂/(2πd); attract if currents parallel
Objectives
  • Apply Biot-Savart to find B for straight wires and loops.
  • Derive B on the axis of a current loop by integration.
  • Calculate force per unit length between parallel wires.
How this can be tested
  • Free-response: derive B at center of a circular arc of angle θ.
  • Biot-Savart: derive B for infinite straight wire by integration.
  • Force: find direction and magnitude between two parallel wires.
Practice before moving on
  • Derive Biot-Savart result for infinite wire by integration.
  • Find B at center of square current loop.

Ampere's Law

Learn on this site
Ampere's Law

∮B·dl = μ₀I_enc; the line integral of B around a closed loop equals μ₀ times the enclosed current. Choose an Amperian loop that simplifies the integral.

Common misconception: Ampere's law applies to all B fields, but it is only useful for finding B when there is sufficient symmetry.

How it can be tested: Apply Ampere's law for straight wires, solenoids, and toroids.

Solenoid

Ideal solenoid (n turns/m, I): B = μ₀nI inside; B = 0 outside. Field is uniform and axial inside.

Common misconception: The solenoid formula B = μ₀nI uses n = turns per meter, not total turns N.

How it can be tested: Calculate B inside a solenoid; explain why B = 0 outside an ideal solenoid.

Toroid

Toroid (N total turns, mean radius R, current I): B = μ₀NI/(2πr) inside; B = 0 outside. B varies with radius inside the toroid.

Common misconception: In a toroid, B is NOT uniform inside — it varies as 1/r. The toroid concentrates flux within the torus.

How it can be tested: Apply Ampere's law to derive B inside a toroid.

Key Vocabulary
  • Ampere's law — ∮B·dl = μ₀I_enc; analogous to Gauss's law for B
  • Amperian loop — Closed path chosen to simplify ∮B·dl
  • Solenoid B — B = μ₀nI inside (uniform); B = 0 outside
  • Turns density n — n = N/L; turns per unit length
  • Toroid B — B = μ₀NI/(2πr) inside; B = 0 outside
Objectives
  • Apply Ampere's law to find B for straight wires, solenoids, and toroids.
  • Identify appropriate Amperian loops for each geometry.
  • Derive B inside a solenoid using Ampere's law.
How this can be tested
  • Free-response: derive B inside a toroidal coil using Ampere's law.
  • Show B = 0 outside an ideal solenoid using Amperian rectangles.
  • Find B inside a thick cylindrical conductor using Ampere's law.
Practice before moving on
  • Derive B for solenoid and toroid from scratch.
  • Apply Ampere's law to a hollow cylindrical conductor.

Magnetic Dipoles and Matter

Learn on this site
Magnetic Dipole Moment

Magnetic dipole moment μ = NIA (A·m²) for a current loop. Torque: τ = μ × B = μB sinθ. PE = -μ·B. Analogous to electric dipole p in electric field.

Common misconception: Magnetic dipole moment μ is the magnetic equivalent of electric dipole p — all formulas are directly analogous.

How it can be tested: Calculate μ, torque, and PE for current loops in magnetic fields.

Magnetic Materials

Diamagnetic materials (χ_m < 0) weakly repel B; paramagnetic (χ_m > 0) weakly attract; ferromagnetic (χ_m >> 0) strongly attract and retain magnetization.

Common misconception: Not all magnets are the same — ferromagnets (iron, nickel) are permanently magnetizable; paramagnets are only temporarily attracted.

How it can be tested: Classify materials by magnetic behavior and explain at the atomic level.

Magnetization and B in Matter

B = μ₀(H + M) where H is the applied field and M is magnetization. For linear magnetic materials: M = χ_m H; B = μ₀(1+χ_m)H = μ₀μ_r H.

Common misconception: H and B are different fields — H is determined by free currents; B includes the contribution from magnetized matter.

How it can be tested: Apply B = μ₀μ_r H to find B in a magnetic material.

Key Vocabulary
  • Magnetic dipole moment μ — μ = NIA; torque τ = μ × B; PE = -μ·B
  • Diamagnetic — χ_m < 0; weakly repelled by B; no permanent moment
  • Paramagnetic — χ_m > 0 (small); weakly attracted to B; temporary alignment
  • Ferromagnetic — χ_m >> 0; permanently magnetized; domains align
  • Relative permeability μ_r — μ_r = 1 + χ_m; B = μ₀μ_r H in linear media
Objectives
  • Calculate magnetic dipole moment, torque, and PE for a current loop.
  • Classify materials as dia-, para-, or ferromagnetic.
  • Apply B = μ₀μ_r H in magnetized materials.
How this can be tested
  • Free-response: find net torque on a current loop in a nonuniform B field.
  • Multiple choice: classify materials by magnetic behavior.
  • Calculate B inside a material from H using given χ_m.
Practice before moving on
  • Practice torque and PE calculations for magnetic dipoles.
  • Compare atomic origins of diamagnetic, paramagnetic, and ferromagnetic behavior.
