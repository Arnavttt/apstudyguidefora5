Position, Velocity, and Acceleration via Calculus

Learn on this site
Derivative Definitions of v and a

Velocity is the derivative of position: v(t) = dx/dt. Acceleration is the derivative of velocity: a(t) = dv/dt = d²x/dt². These are exact, not approximations.

Common misconception: Average velocity (Δx/Δt) is the slope of the secant; instantaneous velocity is the slope of the tangent (derivative).

How it can be tested: Given x(t), find v(t) and a(t) by differentiation.

Integral Definitions

Position change: Δx = ∫v dt. Velocity change: Δv = ∫a dt. The constant of integration is determined by initial conditions.

Common misconception: Indefinite integrals produce functions with a constant C; definite integrals give numerical displacements over an interval.

How it can be tested: Given a(t), integrate to find v(t) and x(t) with initial conditions.

Graphical Interpretation

Slope of x-t graph = v; slope of v-t graph = a; area under v-t graph = Δx; area under a-t graph = Δv.

Common misconception: Area under an a-t graph is Δv, not v — it gives the CHANGE in velocity.

How it can be tested: Sketch or interpret x(t), v(t), a(t) graphs and relate them.

Key Vocabulary
  • x(t) — Position as a function of time
  • v(t) = dx/dt — Instantaneous velocity; slope of x-t graph
  • a(t) = dv/dt — Instantaneous acceleration; slope of v-t graph
  • Δx = ∫v dt — Displacement equals area under v-t graph
  • Initial conditions — Values of x₀ and v₀ used to evaluate constants of integration
Objectives
  • Differentiate position functions to find velocity and acceleration.
  • Integrate acceleration (with initial conditions) to find velocity and position.
  • Relate graphical slopes and areas to kinematic quantities.
How this can be tested
  • Free-response: given x(t) = 3t³ - 5t, find v and a as functions of t.
  • Integration: given a(t) = 6t and v(0) = 2, find v(t) and x(t) with x(0) = 0.
  • Graph interpretation: sketch a-t, v-t, x-t for a given a(t).
Practice before moving on
  • Differentiate and integrate polynomial, sinusoidal, and exponential position functions.
  • Practice finding constants of integration using initial conditions.

1D Kinematics: Constant and Variable Acceleration

Learn on this site
Constant Acceleration Equations

For constant a: v = v₀ + at, x = x₀ + v₀t + ½at², v² = v₀² + 2aΔx. These are derived from integration of a = const.

Common misconception: These equations ONLY apply when acceleration is constant — do not use them if a varies with t.

How it can be tested: Solve 1D kinematics problems analytically and verify with integrals.

Variable Acceleration

When a is not constant, use a(t) directly: integrate to get v(t), integrate again to get x(t). Numerical methods or exact integration may be required.

Common misconception: Constant-acceleration equations give wrong answers for variable acceleration — always check if a is constant.

How it can be tested: Solve problems where a = f(t) using integration.

Jerk

Jerk j = da/dt = d³x/dt³ is the rate of change of acceleration. It appears in smooth motion design (elevators, vehicles).

Common misconception: Jerk is not commonly tested in AP® C but appears in questions about smoothness of motion.

How it can be tested: Differentiate a(t) to find jerk; integrate j(t) to find a(t).

Key Vocabulary
  • Constant acceleration equations — v = v₀ + at; x = x₀ + v₀t + ½at²; v² = v₀² + 2aΔx
  • Free fall — Constant downward acceleration g = 9.8 m/s² near Earth's surface
  • Variable acceleration — a = f(t); requires integration to find v and x
  • Jerk — j = da/dt = d³x/dt³; rate of change of acceleration
  • Integration constant C — Determined by initial conditions v(0) = v₀, x(0) = x₀
Objectives
  • Derive and apply constant-acceleration equations using integration.
  • Solve variable-acceleration problems using definite integrals.
  • Identify when constant-acceleration assumptions break down.
How this can be tested
  • AP® free-response: derive v = v₀ + at by integrating a = const.
  • Problem: a(t) = 6t²; given v(0) = 3, x(0) = 0, find x at t = 2 s.
  • Conceptual: identify which equation applies to which scenario.
Practice before moving on
  • Derive the three constant-acceleration equations from v = v₀ + at.
  • Practice integration problems with polynomial a(t).

2D Kinematics: Projectile and Circular Motion

Learn on this site
Projectile Motion

Decompose into x (constant v_x) and y (a_y = -g). x(t) = x₀ + v_x t; y(t) = y₀ + v_y0 t - ½gt². Horizontal and vertical are independent.

Common misconception: Air resistance is neglected in AP® projectile problems unless stated; horizontal speed is constant throughout.

How it can be tested: Solve for range, max height, time of flight using component equations.

Uniform Circular Motion

Speed is constant but direction changes; centripetal acceleration a_c = v²/r directed toward center. Period T = 2πr/v; frequency f = 1/T.

Common misconception: Centripetal acceleration changes direction of velocity, not its magnitude — the object is accelerating even at constant speed.

How it can be tested: Calculate centripetal acceleration and net centripetal force for circular motion.

Non-Uniform Circular Motion

When speed changes in circular motion, there is both centripetal acceleration (a_c = v²/r, toward center) and tangential acceleration (a_t = dv/dt, along path).

Common misconception: The total acceleration vector is the vector sum of centripetal and tangential components.

How it can be tested: Find magnitude and direction of total acceleration in non-uniform circular motion.

Key Vocabulary
  • Projectile — Object under gravity alone; a_x = 0, a_y = -g
  • Range — Horizontal distance traveled by a projectile
  • Centripetal acceleration — a_c = v²/r directed toward center of circular path
  • Period T — Time for one complete revolution in circular motion
  • Tangential acceleration — a_t = dv/dt; component along the velocity direction in circular motion
Objectives
  • Solve projectile problems by separating x and y components.
  • Calculate centripetal acceleration and associated centripetal force.
  • Find total acceleration vector components in non-uniform circular motion.
How this can be tested
  • Free-response: find range and time of flight for projectile launched at angle θ.
  • Calculation of centripetal force on a car rounding a curve.
  • Non-uniform: find tangential and centripetal acceleration at a given instant.
Practice before moving on
  • Solve projectile problems systematically by listing x and y equations separately.
  • Practice centripetal force problems with various geometries.

Relative Motion and Reference Frames

Learn on this site
Relative Velocity in 1D

Velocity of A relative to B: v_AB = v_A - v_B. Velocities add or subtract depending on direction of observation.

Common misconception: Relative velocity is not symmetric: v_AB = -v_BA.

How it can be tested: Solve 1D relative-velocity problems (boats in rivers, trains).

Relative Velocity in 2D

In 2D, v_AB = v_A - v_B vectorially. For a boat crossing a river, the resultant velocity is the vector sum of boat velocity and current velocity.

Common misconception: To find shortest path across a river, aim the boat upstream at the right angle.

How it can be tested: Solve river-crossing and airplane-wind problems using vector addition.

Non-Inertial Reference Frames

In an accelerating frame, pseudo-forces (fictitious forces) appear. An observer in a non-inertial frame sees fictitious forces opposite to the frame's acceleration.

Common misconception: Pseudo-forces are not real forces — they appear only in accelerating frames, not in inertial frames.

How it can be tested: Describe motion from both inertial and non-inertial frames.

Key Vocabulary
  • Relative velocity — v_AB = v_A - v_B; velocity of A as seen by observer B
  • Inertial frame — Reference frame with no acceleration; Newton's laws hold exactly
  • Non-inertial frame — Accelerating reference frame where fictitious forces appear
  • Pseudo-force — Apparent force in an accelerating frame = -ma_frame
  • River crossing — Classic 2D relative motion problem with cross-current
Objectives
  • Calculate relative velocities in 1D and 2D.
  • Solve boat/plane problems with vector addition of velocities.
  • Describe fictitious forces in non-inertial frames.
How this can be tested
  • Free-response: find the heading for minimum crossing time vs. minimum drift.
  • Multiple choice: determine velocity of one object relative to another.
  • Conceptual: explain why passengers feel pushed back during acceleration.
Practice before moving on
  • Practice river-crossing and airplane-wind vector problems.
  • Compare the same motion described from different reference frames.
