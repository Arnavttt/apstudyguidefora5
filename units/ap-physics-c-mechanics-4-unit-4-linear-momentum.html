Center of Mass

Learn on this site
Center of Mass for Discrete Systems

r_cm = Σ(m_i r_i)/M_total. For a two-body system: x_cm = (m₁x₁ + m₂x₂)/(m₁ + m₂). CM is weighted average of positions.

Common misconception: CM is not necessarily inside the object (e.g., ring, horseshoe).

How it can be tested: Calculate CM coordinates for multiple point masses.

Center of Mass for Continuous Distributions

x_cm = (1/M)∫x dm; use linear density λ, surface density σ, or volume density ρ to express dm.

Common misconception: For uniform density, dm = ρ dV (volume element); for a rod, dm = λ dx.

How it can be tested: Compute CM of uniform rods, triangles, and other shapes using integration.

Motion of the Center of Mass

The CM of an isolated system moves with constant velocity: v_cm = Σ(m_i v_i)/M. External forces change v_cm: F_ext = Ma_cm.

Common misconception: Internal forces between particles do NOT affect the CM's acceleration — only external forces do.

How it can be tested: Apply F_ext = Ma_cm to find CM acceleration in complex systems.

Key Vocabulary
  • Center of mass — r_cm = Σm_i r_i/M; mass-weighted average position
  • Linear density λ — Mass per unit length (kg/m); dm = λ dx for a rod
  • CM velocity — v_cm = p_total/M = Σm_i v_i/M
  • F_ext = Ma_cm — Net external force governs CM acceleration
  • Isolated system CM — CM moves at constant velocity if no external forces act
Objectives
  • Calculate center of mass for point masses and uniform rods.
  • State and apply the theorem that external forces cause CM acceleration.
  • Find CM velocity from individual particle velocities.
How this can be tested
  • Free-response: find CM of L-shaped uniform plate or rod combination.
  • Integration: find CM of a non-uniform rod with λ(x) = λ₀(1 + x/L).
  • Apply F_ext = Ma_cm to a system of connected objects.
Practice before moving on
  • Practice CM calculations for discrete systems of 3–4 masses.
  • Integrate to find CM of a uniform triangle or wedge.

Linear Momentum and Impulse

Learn on this site
Linear Momentum

p = mv; a vector quantity in the direction of velocity. Newton's 2nd law in terms of momentum: F_net = dp/dt (more general than F = ma).

Common misconception: dp/dt = ma only when mass is constant; F = dp/dt handles variable-mass systems (rockets).

How it can be tested: Apply F = dp/dt to situations with changing mass.

Impulse

J = ∫F dt = Δp; impulse equals the area under the F-t graph. For constant force: J = FΔt. Impulse-momentum theorem: J = p_f - p_i.

Common misconception: Impulse is NOT force times time — that is only for constant force; in general, J = ∫F dt.

How it can be tested: Calculate impulse from F-t graphs (area) and from change in momentum.

Conservation of Linear Momentum

If F_ext = 0, then dp/dt = 0 → p_total = constant. Conservation applies separately in each direction when that component of F_ext = 0.

Common misconception: Momentum can be conserved in one direction but not another (e.g., ball bouncing on floor: x conserved, y not).

How it can be tested: Apply momentum conservation to collisions and explosions.

Key Vocabulary
  • Linear momentum p — p = mv; vector in direction of motion (kg·m/s)
  • Impulse J — J = ∫F dt = Δp; change in momentum
  • F = dp/dt — Newton's 2nd law in general form; handles variable-mass problems
  • Impulse-momentum theorem — J = Δp = p_f - p_i
  • Momentum conservation — p_total constant when F_ext = 0
Objectives
  • Calculate impulse from F-t graphs and from momentum change.
  • Apply F = dp/dt to variable-mass problems (rockets).
  • Identify conditions for momentum conservation and apply it.
How this can be tested
  • Free-response: find impulse from a given F(t) and resulting velocity change.
  • Rocket: apply F = dp/dt with varying mass.
  • Collision: conserve momentum (and energy for elastic) to find post-collision velocities.
Practice before moving on
  • Practice impulse calculations using both F-t integration and Δp.
  • Solve rocket problems using the rocket equation qualitatively.

Collisions

Learn on this site
Elastic Collisions

In a perfectly elastic collision, both momentum and kinetic energy are conserved. For 1D elastic collision of m₁ (moving) with m₂ (at rest): v₁' = (m₁-m₂)v₁/(m₁+m₂); v₂' = 2m₁v₁/(m₁+m₂).

Common misconception: In a perfectly elastic collision, the relative speed of approach = relative speed of separation (coefficient of restitution = 1).

How it can be tested: Solve 1D and 2D elastic collisions using momentum and energy conservation.

Inelastic and Perfectly Inelastic Collisions

In a perfectly inelastic collision, objects stick together (maximum KE loss). Partial inelastic: KE lost but objects separate. Only momentum is conserved in all collisions.

Common misconception: In ANY collision (elastic or not), total momentum is conserved; only in elastic collisions is KE also conserved.

How it can be tested: Identify the type of collision and apply appropriate conservation laws.

2D Collisions

In 2D, apply momentum conservation separately for x and y components. At angles, resolve initial and final velocities into components.

Common misconception: In 2D collisions, the objects need not travel along the original axis — resolve all vectors into components.

How it can be tested: Solve 2D glancing collision problems using component momentum conservation.

Key Vocabulary
  • Elastic collision — Both p and KE conserved; coefficient of restitution = 1
  • Inelastic collision — p conserved; KE not conserved; some energy converted to heat
  • Perfectly inelastic — Objects stick together; maximum KE loss
  • Coefficient of restitution — e = |v_rel after|/|v_rel before|; e = 1 elastic, e = 0 perfectly inelastic
  • Glancing collision — 2D collision where object changes direction
Objectives
  • Apply both momentum and energy conservation to elastic collisions.
  • Solve perfectly inelastic collision problems and find KE lost.
  • Set up momentum conservation in x and y for 2D collisions.
How this can be tested
  • Free-response: 2D elastic collision with one mass initially at rest.
  • Calculate KE lost in a perfectly inelastic collision.
  • Derive elastic collision formulas from momentum + energy conservation.
Practice before moving on
  • Derive 1D elastic collision formulas algebraically.
  • Solve 3–4 collision problems including at least one 2D case.

Variable-Mass Systems and Rocket Propulsion

Learn on this site
Rocket Equation

Thrust T = v_ex × |dM/dt|; rocket equation: M(dv/dt) = T - F_drag - Mg. As fuel burns, M decreases and acceleration increases for constant thrust.

Common misconception: The rocket equation requires careful sign conventions: v_exhaust is relative to the rocket.

How it can be tested: Apply rocket thrust equation to find acceleration or velocity change.

Tsiolkovsky Rocket Equation

Δv = v_ex ln(M_i/M_f) for a rocket in gravity-free space. This is derived by integrating M dv = -v_ex dM.

Common misconception: The mass ratio M_i/M_f determines Δv — large fuel fraction means large velocity change.

How it can be tested: Apply the Tsiolkovsky equation to find Δv for given mass ratio and exhaust speed.

Conveyor Belt and Rain Problems

When mass is added to a moving object (rain on a car, sand on a belt), F_ext = dp/dt = d(Mv)/dt. If v is maintained constant, extra force is required to compensate for increased mass.

Common misconception: Force to maintain constant speed on a conveyor belt with added mass: F = v × (dm/dt) — additional force needed for momentum of added mass.

How it can be tested: Apply F = dp/dt to variable-mass problems other than rockets.

Key Vocabulary
  • Thrust — F_thrust = v_ex × |dM/dt|; force from ejected propellant
  • Tsiolkovsky equation — Δv = v_ex ln(M_i/M_f) for rocket in free space
  • Mass ratio — M_i/M_f; determines Δv; larger ratio → greater velocity change
  • Exhaust velocity — v_ex; speed of exhaust relative to rocket; determines efficiency
  • Conveyor belt problem — Variable-mass system: F = v(dm/dt) to maintain constant speed
Objectives
  • Calculate thrust force from exhaust velocity and mass flow rate.
  • Apply Tsiolkovsky equation to find Δv or required mass ratio.
  • Solve conveyor belt and rain-on-cart problems using F = dp/dt.
How this can be tested
  • Free-response: rocket in gravity — write and analyze equation of motion.
  • Tsiolkovsky: find mass ratio required for Δv = 8 km/s with v_ex = 3 km/s.
  • Conveyor belt: find motor power to maintain belt speed with added sand.
Practice before moving on
  • Derive the Tsiolkovsky equation from M dv = -v_ex dM.
  • Practice rocket problems with and without gravity.
