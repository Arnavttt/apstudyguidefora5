Rotational Kinematics

Learn on this site
Angular Position, Velocity, and Acceleration

θ = angular position (rad); ω = dθ/dt (rad/s); α = dω/dt = d²θ/dt² (rad/s²). All directly analogous to linear kinematics with x→θ, v→ω, a→α.

Common misconception: Angular quantities are not vectors in 3D (they are for ω and α), but in AP® C problems they are treated as scalars with sign indicating direction.

How it can be tested: Differentiate and integrate angular position functions; apply rotational kinematic equations.

Constant Angular Acceleration Equations

For constant α: ω = ω₀ + αt; θ = θ₀ + ω₀t + ½αt²; ω² = ω₀² + 2αΔθ. These mirror the linear equations.

Common misconception: Do NOT use these equations if α is a function of time — integrate α(t) instead.

How it can be tested: Solve rotational problems with constant α analogously to linear kinematics.

Relating Linear and Rotational Quantities

For a point at radius r: s = rθ; v_t = rω; a_t = rα; a_c = v²/r = rω². Rolling without slipping: v_cm = rω; a_cm = rα.

Common misconception: Tangential and centripetal accelerations are distinct: a_t = rα (changing speed); a_c = rω² (changing direction).

How it can be tested: Convert between linear and angular quantities using radius.

Key Vocabulary
  • θ (angular position) — Angle measured in radians from a reference line
  • ω = dθ/dt — Angular velocity (rad/s)
  • α = dω/dt — Angular acceleration (rad/s²)
  • v_t = rω — Tangential speed of a point at radius r
  • Rolling condition — v_cm = rω for rolling without slipping
Objectives
  • Differentiate θ(t) to find ω(t) and α(t).
  • Apply constant-α rotational equations.
  • Convert between tangential and angular quantities using r.
How this can be tested
  • Free-response: find α(t) by differentiating given ω(t).
  • Constant α: find angle rotated and angular speed after time t.
  • Rolling: find speed of a point on the rim of a rolling wheel.
Practice before moving on
  • Derive constant-α equations by integrating α = const.
  • Practice linear-rotational quantity conversions for various radii.

Torque and Newton's Second Law for Rotation

Learn on this site
Torque

τ = r × F = rF sinθ (N·m). Torque is the rotational analog of force; it depends on force, lever arm, and angle. Choose the correct pivot point.

Common misconception: Torque is NOT the same as force — it includes the lever arm (perpendicular distance from pivot to line of action).

How it can be tested: Calculate torques from multiple forces about a specified pivot point.

Newton's Second Law for Rotation

τ_net = Iα, where I is the moment of inertia (kg·m²) and α is angular acceleration. Analogous to F_net = ma.

Common misconception: I is not constant if mass distribution changes; τ_net must include all torques about the same axis.

How it can be tested: Apply τ = Iα to rotating systems; combine with translational F = ma for rolling objects.

Static Equilibrium

For a rigid body in static equilibrium: ΣF = 0 and Στ = 0 about any point. Choose the pivot to eliminate unknown forces and simplify calculation.

Common misconception: The condition Στ = 0 must be satisfied about ANY axis — choosing wisely can eliminate unknowns.

How it can be tested: Solve static equilibrium problems (beams, ladders, levers).

Key Vocabulary
  • Torque τ — τ = rF sinθ = r⊥F = rF⊥; rotational effect of a force
  • Lever arm — Perpendicular distance from pivot to line of action of force
  • Moment of inertia I — I = Σm_i r_i² or ∫r² dm; rotational inertia
  • τ_net = Iα — Rotational Newton's 2nd law
  • Static equilibrium — ΣF = 0 and Στ = 0; no translation or rotation
Objectives
  • Calculate torque for forces at various angles about a given pivot.
  • Apply τ_net = Iα to find angular acceleration.
  • Solve static equilibrium problems using ΣF = 0 and Στ = 0.
How this can be tested
  • Free-response: beam supported at two points — find both normal forces.
  • Rotation: find α for a disk given net torque and I.
  • Ladder leaning against wall: find minimum friction coefficient.
Practice before moving on
  • Practice torque calculations for multiple forces.
  • Solve at least 3 static equilibrium problems (beam, ladder, see-saw).

Moment of Inertia

Learn on this site
Discrete System Moment of Inertia

I = Σm_i r_i² for point masses. I depends on both mass and distribution — farther from axis means larger I.

Common misconception: I depends on which axis is chosen — always specify the rotation axis.

How it can be tested: Calculate I for systems of point masses about a given axis.

Continuous Distribution: Integration

I = ∫r² dm; use geometric relationships to express dm in terms of the integration variable. Common results: disk = ½MR², rod about CM = (1/12)ML², sphere = (2/5)MR².

Common misconception: The moment of inertia of a solid disk is ½MR², NOT MR² — the mass is distributed at various distances, not all at the rim.

How it can be tested: Derive moments of inertia for rods and disks using integration.

Parallel Axis Theorem

I_parallel = I_cm + Md², where d is the distance from CM to the new axis. Allows calculation of I about any axis from the CM result.

Common misconception: The parallel axis theorem ADDS Md² to I_cm — it never gives a smaller I than the CM result.

How it can be tested: Apply the parallel axis theorem to find I about axes not through the CM.

Key Vocabulary
  • Moment of inertia I — I = ∫r² dm; measure of rotational inertia (kg·m²)
  • Parallel axis theorem — I = I_cm + Md²; I about any axis from CM result
  • I for common shapes — Disk: ½MR²; rod (CM): (1/12)ML²; solid sphere: (2/5)MR²
  • Radius of gyration k — I = Mk²; effective radius at which all mass could be concentrated
  • Hollow vs. solid cylinder — Hollow shell I = MR²; solid I = ½MR² (mass is closer to axis)
Objectives
  • Calculate I for point mass systems.
  • Derive I for a uniform rod and disk using integration.
  • Apply the parallel axis theorem for off-CM axes.
How this can be tested
  • Free-response: derive I for a thin rod about one end using integration.
  • Calculate I for a system of a disk plus attached masses.
  • Apply parallel axis theorem to find I for a disk about its rim.
Practice before moving on
  • Derive I_rod (end) from I_rod (CM) using the parallel axis theorem.
  • Verify I_disk = ½MR² using integration.

Angular Momentum and Rotational Energy

Learn on this site
Angular Momentum

L = Iω for a rigid body; L = r × p = r × mv for a particle. Conservation: if τ_ext = 0, then L = constant.

Common misconception: Angular momentum of a point mass is not zero even for straight-line motion — it is r × mv about any point not on the line of motion.

How it can be tested: Calculate L for rotating bodies and for particles in linear or circular motion.

Conservation of Angular Momentum

When τ_ext = 0: L_i = L_f → I_i ω_i = I_f ω_f. Applies to collisions involving rotation, spinning skaters, etc.

Common misconception: Angular momentum is conserved for the system — internal forces and torques do not change the total.

How it can be tested: Apply L conservation to collisions with rotating objects and changing moment of inertia.

Rotational Kinetic Energy

KE_rot = ½Iω²; rolling objects have both KE_trans = ½mv_cm² and KE_rot = ½Iω². Total KE = ½mv² + ½Iω².

Common misconception: A rolling object has MORE kinetic energy than a sliding object of the same mass and speed — some energy is in rotation.

How it can be tested: Apply energy conservation to rolling objects including rotational KE.

Key Vocabulary
  • Angular momentum L — L = Iω (rigid body); L = r × p (particle); units: kg·m²/s
  • L conservation — L = const when τ_ext = 0; I_i ω_i = I_f ω_f
  • Rotational KE — KE_rot = ½Iω²; must add to translational KE for rolling
  • Total rolling KE — KE_total = ½mv² + ½Iω² = ½mv²(1 + I/(mr²))
  • Rotational work — W_rot = τΔθ; work done by a torque over an angular displacement
Objectives
  • Calculate angular momentum for rotating rigid bodies and particles.
  • Apply L conservation to solve rotation + collision problems.
  • Include rotational KE in energy conservation for rolling objects.
How this can be tested
  • Free-response: ball rolling down a ramp — find speed using energy with rotation.
  • Collision: mass lands on a rotating disk — find final ω.
  • L conservation: spinning skater changes I; find final ω.
Practice before moving on
  • Practice energy conservation with rolling for several shapes.
  • Solve angular momentum collision problems.
