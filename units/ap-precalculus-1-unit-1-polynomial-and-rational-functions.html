Polynomial Functions and Their Behavior

Learn on this site
Degree and Leading Coefficient

The degree is the highest power of x. The leading coefficient determines end behavior with the degree: even degree, positive LC → both ends up (↑↑); even, negative → both down (↓↓); odd, positive → down-up (↓↑); odd, negative → up-down (↑↓).

Common misconception: End behavior depends only on degree and leading coefficient — not on any other terms.

How it can be tested: State end behavior from the leading term; match polynomial graphs to their degree and LC.

Zeros, Multiplicity, and Graphs

A zero of multiplicity 1 crosses the x-axis; multiplicity 2 (even) touches and bounces; multiplicity 3 (odd, ≥3) crosses with inflection. Total degree = sum of multiplicities.

Common misconception: Multiplicity tells you the behavior at each zero — multiplicity 2 does NOT always bounce at 45°, it touches smoothly.

How it can be tested: Determine zeros and their multiplicities from a factored polynomial; sketch behavior at each zero.

Average Rate of Change

ARC = [f(b) − f(a)] / (b − a): the slope of the secant line from a to b. Measures how fast a function changes over an interval. AP® Precalculus tests ARC for polynomial, exponential, and log functions.

Common misconception: ARC is not the instantaneous rate of change (derivative) — it is the average over an interval. These are not the same for nonlinear functions.

How it can be tested: Calculate and interpret ARC for polynomials; compare ARC across different intervals.

Key Vocabulary
  • degree — Highest power of x in the polynomial; determines end behavior category
  • leading coefficient — Coefficient of the highest-degree term; sign determines up/down end behavior
  • zero (root) — x-value where f(x) = 0; where the graph crosses or touches the x-axis
  • multiplicity — How many times a factor (x−r) appears; determines crossing vs bouncing at r
  • average rate of change — (f(b)−f(a))/(b−a); slope of the secant line; average change per unit x
Objectives
  • Determine end behavior from the leading term of a polynomial.
  • Identify zeros and their multiplicities from a factored polynomial.
  • Calculate and interpret average rate of change over an interval.
How this can be tested
  • MCQ/FRQ: state end behavior from leading term; match graph to description.
  • FRQ: find zeros and multiplicities; sketch graph behavior.
  • FRQ: calculate ARC and explain what it represents in context.
Practice before moving on
  • Sketch polynomial graphs using end behavior, zeros, and multiplicities.
  • Calculate ARC for a polynomial on different intervals and compare.

Rational Functions

Learn on this site
Vertical and Horizontal Asymptotes

VA: x-values where denominator = 0 (and numerator ≠ 0). HA: compare degrees of numerator (n) and denominator (d). n < d → y = 0; n = d → y = leading coefficients ratio; n > d → no HA (slant instead).

Common misconception: A zero of the denominator is NOT always a vertical asymptote — it may be a hole if the factor cancels with a numerator factor.

How it can be tested: Find VAs, HAs, and holes; explain the difference between holes and asymptotes.

Holes and Removable Discontinuities

A hole occurs when the same factor appears in both numerator and denominator. Cancel the common factor to find the simplified form; hole is at that x-value in the original function.

Common misconception: Holes are NOT visible as gaps on a typical graphing calculator screen — you must find them algebraically.

How it can be tested: Find holes by factoring and canceling; evaluate the y-coordinate of a hole.

Graphing Rational Functions

Procedure: factor → find domain → find intercepts → find asymptotes → find holes → analyze sign and sketch. End behavior follows the simplified rational expression.

Common misconception: The graph approaches but never touches asymptotes for large |x|, but the function CAN cross its HA for smaller x-values.

How it can be tested: Sketch a rational function using all key features; interpret the graph in context.

Key Vocabulary
  • vertical asymptote — x = a where denominator = 0 and numerator ≠ 0; function approaches ±∞
  • horizontal asymptote — y = L that the function approaches as x → ±∞; depends on degree comparison
  • hole (removable discontinuity) — Open point on graph where a common factor cancels; function undefined there
  • slant (oblique) asymptote — When numerator degree = denominator degree + 1; found by polynomial long division
  • domain of rational function — All real x except zeros of the denominator
Objectives
  • Find all asymptotes and holes of a rational function.
  • Determine the horizontal asymptote by comparing numerator and denominator degrees.
  • Sketch a rational function using all key features.
How this can be tested
  • FRQ: find domain, intercepts, asymptotes, and holes; describe end behavior.
  • MCQ: identify the type of asymptote given degree comparison.
  • FRQ: explain why a feature is a hole vs a vertical asymptote.
Practice before moving on
  • Complete a sign analysis to sketch a rational function.
  • Compare two rational functions: one with HA y=0, one with HA y=2.

Polynomial Inequalities and Systems

Learn on this site
Solving Polynomial Inequalities

Find zeros → set up sign chart → test intervals → identify where inequality is satisfied. Include endpoints if ≤ or ≥; exclude for strict inequalities. Use factored form for sign analysis.

Common misconception: When multiplying or dividing both sides of an inequality by a negative, flip the inequality sign. This is a common algebra error.

How it can be tested: Solve polynomial and rational inequalities using sign charts; write solution in interval notation.

Polynomial Division and Remainder Theorem

Remainder Theorem: f(a) = remainder when f(x) is divided by (x−a). Factor Theorem: (x−a) is a factor iff f(a) = 0. Synthetic division is efficient for linear divisors.

Common misconception: The Remainder Theorem gives the function value at x=a without dividing — a powerful shortcut for evaluating and factoring.

How it can be tested: Use synthetic division to find the remainder; apply the Factor Theorem to test potential zeros.

Rational Root Theorem

Rational zeros of f(x) must have the form p/q where p divides the constant term and q divides the leading coefficient. Provides a finite list of candidates to test.

Common misconception: The Rational Root Theorem lists CANDIDATES — not all candidates are actual zeros. You must test each one.

How it can be tested: List rational zero candidates; use synthetic division to confirm zeros; factor completely.

Key Vocabulary
  • sign chart — Table or number line showing where a function/expression is positive or negative
  • Remainder Theorem — f(a) equals the remainder when f(x) is divided by (x−a)
  • Factor Theorem — (x−a) is a factor of f(x) if and only if f(a) = 0
  • synthetic division — Shorthand method for dividing a polynomial by a linear factor (x−a)
  • Rational Root Theorem — Rational zeros have form p/q: p|constant term, q|leading coefficient
Objectives
  • Solve polynomial inequalities using sign charts and interval notation.
  • Apply the Remainder and Factor Theorems to evaluate and factor polynomials.
  • Use the Rational Root Theorem to find possible rational zeros.
How this can be tested
  • FRQ: find all zeros of a cubic using Rational Root Theorem and synthetic division.
  • MCQ: apply the Remainder Theorem to find f(a) without dividing.
  • FRQ: solve a polynomial inequality and write the solution set.
Practice before moving on
  • Practice synthetic division on several examples.
  • Solve (x−1)(x+2)(x−3) ≥ 0 using a sign chart.

Complex Numbers and Polynomial Completeness

Learn on this site
Complex Numbers

i = √(−1); i² = −1. Complex number: a + bi. Add/subtract: combine real and imaginary parts. Multiply: FOIL then replace i² with −1. Conjugate of a+bi is a−bi. Divide by multiplying numerator and denominator by the conjugate.

Common misconception: i² = −1, not 1. This is the defining property and is used every time you multiply complex numbers.

How it can be tested: Perform arithmetic on complex numbers; find conjugates; divide complex numbers.

Complex Zeros and Conjugate Pairs

Fundamental Theorem of Algebra: a degree-n polynomial has exactly n zeros (in ℂ, counting multiplicity). Complex zeros of real polynomials always come in conjugate pairs: if a+bi is a zero, so is a−bi.

Common misconception: A real polynomial of odd degree must have at least one real zero — complex zeros come in pairs, so an odd total requires at least one real.

How it can be tested: Find a polynomial with given complex zeros; apply conjugate pair theorem; count zeros.

Writing Polynomials from Zeros

Given zeros (real and complex), construct the polynomial: multiply the factors (x − zero) for each zero. For complex conjugate pairs: (x−(a+bi))(x−(a−bi)) = x² − 2ax + (a²+b²), a quadratic with real coefficients.

Common misconception: Each complex zero contributes a quadratic factor (not two separate complex linear factors) when writing a real-coefficient polynomial.

How it can be tested: Construct a polynomial with specified real and complex zeros; verify by expanding.

Key Vocabulary
  • imaginary unit i — i = √(−1); i² = −1; foundation of complex number system
  • complex number — Number of the form a + bi where a (real part) and b (imaginary part) are real
  • complex conjugate — Conjugate of a+bi is a−bi; product (a+bi)(a−bi) = a²+b² (real)
  • Fundamental Theorem of Algebra — Every degree-n polynomial has exactly n zeros in ℂ counting multiplicity
  • conjugate pair theorem — Complex zeros of real-coefficient polynomials come in conjugate pairs
Objectives
  • Perform arithmetic with complex numbers.
  • Apply the conjugate pair theorem to find all zeros of a real polynomial.
  • Write a polynomial with specified real and complex zeros.
How this can be tested
  • FRQ: given some zeros (including complex), find all zeros and write the polynomial.
  • MCQ: identify a complex arithmetic result.
  • FRQ: explain why complex zeros come in conjugate pairs for real polynomials.
Practice before moving on
  • Practice dividing complex numbers using conjugates.
  • Find all zeros of x⁴ − x² − 12, including complex ones if they exist.
