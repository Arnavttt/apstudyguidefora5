The Unit Circle and Trig Functions

Learn on this site
Unit Circle Definition

Unit circle: radius 1 centered at origin. For angle θ (in radians), the point on the unit circle is (cos θ, sin θ). Key angles: 0, π/6, π/4, π/3, π/2 and their reflections in all four quadrants.

Common misconception: Sine is the y-coordinate and cosine is the x-coordinate on the unit circle — NOT the other way around.

How it can be tested: Find exact values of sin and cos for special angles; determine quadrant signs.

Trig Functions and Their Properties

sin θ: odd function (range [−1,1]); cos θ: even function (range [−1,1]); tan θ = sin/cos (range all reals, undefined where cos=0). Period of sin and cos: 2π; period of tan: π.

Common misconception: Tangent is undefined at θ = π/2 + nπ (where cosine is zero) — not at θ = nπ.

How it can be tested: State domain, range, period, and key properties of sin, cos, and tan.

Radians vs Degrees

Radian measure: angle whose arc length equals radius. π radians = 180°. Convert: radians = degrees × (π/180); degrees = radians × (180/π). AP® Precalculus works primarily in radians.

Common misconception: Calculators must be in RADIAN mode for AP® Precalculus. Forgetting to switch is a common error.

How it can be tested: Convert between radians and degrees; evaluate trig functions in radian measure.

Key Vocabulary
  • unit circle — Circle of radius 1 centered at origin; point at angle θ is (cosθ, sinθ)
  • radian — Angle measure equal to arc length / radius; 2π radians = 360°
  • sin θ — y-coordinate on unit circle at angle θ; range [−1,1]; period 2π; odd function
  • cos θ — x-coordinate on unit circle at angle θ; range [−1,1]; period 2π; even function
  • tan θ — sin θ / cos θ; range (−∞,∞); period π; undefined where cos θ = 0
Objectives
  • State exact values of sin, cos, and tan at special angles.
  • Convert between radians and degrees.
  • State domain, range, and period of each trig function.
How this can be tested
  • MCQ: evaluate sin(5π/6) or cos(7π/4) using the unit circle.
  • FRQ: find the exact value and explain using the unit circle.
  • MCQ: identify the quadrant of an angle given its trig signs.
Practice before moving on
  • Memorize or derive the unit circle values for all special angles.
  • Convert 210° and 3π/4 between degrees and radians.

Graphing Sinusoidal Functions

Learn on this site
Amplitude and Period

f(x) = A sin(Bx + C) + D. |A| = amplitude (half the vertical range). Period = 2π/|B|. Midline: y = D. Phase shift: −C/B (horizontal shift). AP® Precalculus tests all four parameters.

Common misconception: Amplitude is |A|, not A — it is always positive. A = −3 gives amplitude 3, not −3.

How it can be tested: Determine amplitude, period, midline, and phase shift from an equation or graph.

Phase Shift and Vertical Shift

Phase shift = −C/B shifts the graph horizontally. Vertical shift D moves the midline. Transformations combine: sketch the parent sin/cos, then apply shifts and stretches.

Common misconception: Phase shift is −C/B, not C. For sin(2x−π), B=2, C=−π: phase shift = −(−π)/2 = π/2 (right).

How it can be tested: Determine the phase shift from f(x) = A sin(Bx + C) + D; sketch the transformed graph.

Modeling with Sinusoidal Functions

Real-world periodic phenomena (tides, temperature, sound) modeled by A sin(Bx+C)+D. Given max, min, and period: A = (max−min)/2; D = (max+min)/2; B = 2π/period.

Common misconception: The midline (D) is not the minimum — it is the average of max and min.

How it can be tested: Build a sinusoidal model from given real-world data; interpret amplitude and period in context.

Key Vocabulary
  • amplitude — |A| in A sin(Bx+C)+D; half the vertical range (max−min)/2; always positive
  • period — 2π/|B|; horizontal length of one complete cycle
  • phase shift — −C/B; horizontal translation; positive → right, negative → left
  • midline — y = D; horizontal line equidistant from max and min; vertical shift of parent
  • sinusoidal function — Function of the form A sin(Bx+C)+D or A cos(Bx+C)+D
Objectives
  • Identify amplitude, period, phase shift, and midline from an equation.
  • Write a sinusoidal equation from given graph features.
  • Build a sinusoidal model from real-world max/min/period data.
How this can be tested
  • FRQ: find all parameters from a graph and write the equation.
  • MCQ: identify amplitude or period from a given equation.
  • FRQ: model real-world periodic data with a sinusoidal function.
Practice before moving on
  • Graph f(x) = 3 sin(2x − π) + 1; label all key features.
  • Write a cosine model for ocean tides with max 4m, min −2m, period 12 hours.

Trigonometric Identities and Inverse Functions

Learn on this site
Fundamental Identities

Pythagorean: sin²θ + cos²θ = 1 → also 1+tan²θ = sec²θ; 1+cot²θ = csc²θ. Reciprocal: csc=1/sin, sec=1/cos, cot=1/tan. Quotient: tan=sin/cos. These are verified for all θ where defined.

Common misconception: Identities hold for all valid θ (not specific values). An equation true for only some θ is a conditional equation, not an identity.

How it can be tested: Use fundamental identities to simplify trig expressions and verify identities.

Sum and Difference, Double Angle

sin(A±B) = sinA cosB ± cosA sinB. cos(A±B) = cosA cosB ∓ sinA sinB. Double angle: sin(2A)=2sinA cosA; cos(2A)=cos²A−sin²A=1−2sin²A=2cos²A−1.

Common misconception: cos(2A) has three equivalent forms — choose the one that makes simplification easiest based on context.

How it can be tested: Apply sum/difference and double angle identities to simplify or evaluate expressions.

Inverse Trigonometric Functions

sin⁻¹(x) (arcsin): range [−π/2, π/2]. cos⁻¹(x) (arccos): range [0, π]. tan⁻¹(x) (arctan): range (−π/2, π/2). These restricted ranges ensure the inverse is a function.

Common misconception: sin⁻¹(sin(x)) ≠ x in general — only for x in [−π/2, π/2]. The restricted range matters.

How it can be tested: Evaluate inverse trig functions; solve trig equations using inverse functions with correct quadrants.

Key Vocabulary
  • Pythagorean identity — sin²θ + cos²θ = 1; derived from unit circle definition; three equivalent forms
  • double angle formulas — sin(2A)=2sinAcosA; cos(2A)=cos²A−sin²A (three equivalent forms)
  • arcsin (sin⁻¹) — Inverse sine; range [−π/2, π/2]; outputs angle whose sine equals the input
  • arccos (cos⁻¹) — Inverse cosine; range [0, π]; restricted to make it a function
  • restricted domain — Domain limitation applied to trig functions to make their inverses functions
Objectives
  • Apply the Pythagorean and other fundamental identities.
  • Use sum/difference and double angle formulas.
  • Evaluate inverse trig functions and interpret their restricted ranges.
How this can be tested
  • FRQ: simplify a trig expression using identities.
  • MCQ: evaluate sin⁻¹ or cos⁻¹ for a specific value.
  • FRQ: solve a trig equation over a specified interval.
Practice before moving on
  • Verify the identity: (1−cos²θ)/sinθ = sinθ.
  • Solve sin(2x) = cos(x) for x ∈ [0, 2π).

Polar Coordinates

Learn on this site
Polar Coordinate System

Point (r, θ): r is distance from origin (pole), θ is angle from positive x-axis. Convert: x = rcosθ; y = rsinθ; r² = x²+y²; tanθ = y/x. Multiple representations: (r,θ) = (r,θ+2π) = (−r,θ+π).

Common misconception: Polar coordinates are NOT unique — the same point has infinitely many representations. This is different from Cartesian coordinates.

How it can be tested: Convert between polar and Cartesian; identify multiple polar representations of a point.

Polar Equations and Graphs

Common polar curves: circle r=a; rose r=cos(nθ); limaçon r=a+bcosθ; cardioid r=1+cosθ (limaçon with a=b). Number of petals in r=cos(nθ): n petals if n odd; 2n petals if n even.

Common misconception: The number of petals in a rose curve depends on whether n is odd or even — different formula for each.

How it can be tested: Sketch common polar curves; identify curve type from a polar equation.

Converting and Analyzing Polar Curves

Convert polar to Cartesian by substituting r²=x²+y², rcosθ=x, rsinθ=y. Symmetry tests: if f(θ) = f(−θ), symmetric about polar axis; if f(π−θ)=f(θ), symmetric about θ=π/2.

Common misconception: Symmetry tests in polar are sufficient but not necessary — a curve may be symmetric even if the test fails.

How it can be tested: Convert a polar equation to Cartesian; test and apply polar symmetry.

Key Vocabulary
  • polar coordinates — (r, θ): r = radial distance, θ = angle from positive x-axis
  • pole — Origin (0,0) of the polar coordinate system
  • cardioid — Polar curve r = a(1+cosθ) or r = a(1+sinθ); heart-shaped
  • rose curve — r = a cos(nθ); n petals if n odd; 2n petals if n even
  • limaçon — r = a + b cosθ; shape depends on ratio a/b: loop, cardioid, or dimpled
Objectives
  • Convert between polar and Cartesian coordinates.
  • Identify and sketch common polar curves.
  • Test polar curves for symmetry about the polar axis or θ=π/2.
How this can be tested
  • FRQ: convert a polar equation to Cartesian; identify the shape.
  • MCQ: identify the number of petals in a rose curve.
  • FRQ: find all polar representations of a given point.
Practice before moving on
  • Graph r = 2 + 2cosθ (cardioid) by creating a table of (r,θ) values.
  • Convert r = 4cosθ to Cartesian and identify the shape.
