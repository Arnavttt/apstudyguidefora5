Parametric Equations

Learn on this site
Introduction to Parametric Equations

Parametric equations express x and y as functions of a parameter t: x=f(t), y=g(t). The parameter t often represents time. Eliminates t by solving one equation for t and substituting.

Common misconception: The orientation (direction of travel as t increases) is part of the parametric curve — the same Cartesian curve can have different orientations from different parameterizations.

How it can be tested: Trace a parametric curve by evaluating at key t-values; eliminate t to find the Cartesian equation.

Graphing and Interpreting Parametric Curves

To graph: make a table of (t, x, y) → plot (x,y). Eliminating t gives the underlying shape; restrictions on t may limit the portion of the curve. Parametric form is natural for projectile motion: x=v₀t cosα, y=v₀t sinα − (1/2)gt².

Common misconception: Eliminating t may lose information about direction and range — always consider the domain of t.

How it can be tested: Graph parametric curves; interpret the parameter as time in motion problems.

Parametric to Cartesian Conversion

For trig parameterizations: x = cosθ, y = sinθ → x²+y²=1 (unit circle). Use Pythagorean identity. For x=at+b: solve t=(x−b)/a, substitute into y equation. Verify the restriction on t matches the desired curve portion.

Common misconception: A parameterization can trace only PART of a curve if t has a limited range — the Cartesian equation may be broader.

How it can be tested: Convert parametric equations to Cartesian form; identify any restrictions.

Key Vocabulary
  • parameter — Variable (often t or θ) that defines both x and y in parametric equations
  • parametric equations — Pair x=f(t), y=g(t) describing a curve by position at each parameter value
  • eliminating the parameter — Solving for t in one equation and substituting to get a Cartesian relation
  • orientation — Direction of travel along a parametric curve as t increases
  • projectile motion — x=v₀cosα·t; y=v₀sinα·t−½gt²; natural parametric form with t=time
Objectives
  • Graph parametric curves using tables of t-values.
  • Eliminate the parameter to find the Cartesian equation.
  • Interpret parametric equations in the context of motion.
How this can be tested
  • FRQ: eliminate the parameter to find a Cartesian equation.
  • MCQ: identify the Cartesian shape from parametric equations.
  • FRQ: find position, velocity, or time from a parametric motion model.
Practice before moving on
  • Graph x = 2cosθ, y = 3sinθ and identify the shape.
  • Eliminate t from x = t+1, y = t² to find the Cartesian equation.

Vectors in the Plane

Learn on this site
Vector Basics

A vector has magnitude and direction. Notation: v = ⟨a,b⟩ or v = ai + bj. Magnitude: |v| = √(a²+b²). Direction angle: tanθ = b/a. Vector addition: ⟨a,b⟩+⟨c,d⟩=⟨a+c,b+d⟩. Scalar multiplication: k⟨a,b⟩=⟨ka,kb⟩.

Common misconception: Vectors are NOT points — they represent displacement. Two vectors with the same components are equal regardless of where they start.

How it can be tested: Compute vector magnitude, direction angle; perform vector addition and scalar multiplication.

Unit Vectors and Components

Unit vector: magnitude 1. Unit vector in direction of v: û = v/|v|. i = ⟨1,0⟩; j = ⟨0,1⟩. Any vector: v = ai + bj. Component form from magnitude and angle: a = |v|cosθ, b = |v|sinθ.

Common misconception: A unit vector has magnitude 1, not component 1. ⟨3,4⟩ has magnitude 5; unit vector = ⟨3/5, 4/5⟩.

How it can be tested: Find unit vectors; express vectors in component form from magnitude and angle.

Dot Product and Applications

Dot product: u·v = u₁v₁+u₂v₂ = |u||v|cosθ. Orthogonal vectors: u·v = 0. Projection of u onto v: proj_v u = (u·v/|v|²)v. Work: W = F·d.

Common misconception: Dot product is a scalar (number), not a vector. This distinguishes it from cross product (which is a vector in 3D).

How it can be tested: Compute dot products; determine if vectors are orthogonal; find the angle between vectors.

Key Vocabulary
  • vector — Mathematical object with magnitude and direction; represented as ⟨a,b⟩
  • magnitude |v| — √(a²+b²) for v=⟨a,b⟩; length of the vector arrow
  • unit vector — Vector of magnitude 1; found by dividing v by |v|
  • dot product — u·v = u₁v₁+u₂v₂; scalar result; equals |u||v|cosθ
  • orthogonal vectors — Vectors whose dot product is 0; perpendicular; angle between them is 90°
Objectives
  • Compute vector magnitude and direction angle.
  • Perform vector addition, subtraction, and scalar multiplication.
  • Compute dot products and use them to find angles or test orthogonality.
How this can be tested
  • FRQ: find magnitude and direction of a vector sum.
  • MCQ: determine if two vectors are orthogonal.
  • FRQ: find the work done by a force vector over a displacement.
Practice before moving on
  • Find the unit vector in the direction of ⟨3,−4⟩.
  • Find the angle between u=⟨2,1⟩ and v=⟨1,3⟩ using dot product.

Matrices and Inverse Functions

Learn on this site
Matrix Operations

Matrix: rectangular array of numbers. Addition/subtraction: element-wise, must have same dimensions. Multiplication: (A×B)ᵢⱼ = row i of A · column j of B; need inner dimensions to match. Not commutative: AB ≠ BA generally.

Common misconception: Matrix multiplication requires columns of A = rows of B. A 2×3 times a 3×2 gives 2×2; 3×2 times 2×3 gives 3×3.

How it can be tested: Perform matrix multiplication; identify when dimensions are compatible.

Inverse Matrices and Systems

For 2×2 matrix A = [[a,b],[c,d]]: det(A) = ad−bc; A⁻¹ = (1/det)[[d,−b],[−c,a]] if det≠0. Solve AX = B by X = A⁻¹B. If det = 0, the matrix is singular (no unique solution).

Common misconception: A⁻¹B ≠ BA⁻¹ because matrix multiplication is not commutative. Always multiply on the correct side.

How it can be tested: Find the inverse of a 2×2 matrix; use it to solve a system of equations.

Inverse Functions

f⁻¹ is the inverse of f if f(f⁻¹(x)) = x and f⁻¹(f(x)) = x. To find: replace f(x) with y, swap x and y, solve for y. The inverse exists only if f is one-to-one (passes horizontal line test).

Common misconception: f⁻¹(x) is NOT 1/f(x). The notation f⁻¹ means the inverse function, not the reciprocal.

How it can be tested: Find the inverse of a function algebraically; verify by composition; state the domain.

Key Vocabulary
  • matrix — Rectangular array of numbers organized in rows and columns
  • determinant (2×2) — det[[a,b],[c,d]] = ad−bc; non-zero means inverse exists
  • matrix inverse A⁻¹ — Matrix such that AA⁻¹ = A⁻¹A = I (identity matrix); exists only if det ≠ 0
  • inverse function f⁻¹ — Function that reverses f; f(f⁻¹(x))=x; found by swapping x and y
  • one-to-one function — Function passing horizontal line test; each output corresponds to one input; invertible
Objectives
  • Multiply matrices and check dimension compatibility.
  • Find the inverse of a 2×2 matrix and use it to solve a system.
  • Find and verify the inverse of a function; state its domain.
How this can be tested
  • FRQ: find A⁻¹ and use it to solve AX = B.
  • MCQ: determine if two matrices can be multiplied given their dimensions.
  • FRQ: find the inverse of f(x) = (2x+1)/(x−3) and state its domain.
Practice before moving on
  • Find the inverse of [[3,1],[5,2]] and verify by computing A·A⁻¹.
  • Find f⁻¹(x) for f(x) = 3x−7 and verify f(f⁻¹(x)) = x.

Find the inverse of A = [[2,3],[1,4]].
  • A) Elements: (1/5)[[4,−3],[−1,2]] = [[0.8,−0.6],[−0.2,0.4]]
  • B) det(A) = 2(4)−3(1) = 8−3 = 5
  • C) A⁻¹ = (1/5)[[4,−3],[−1,2]]
  • D) Verify: A·A⁻¹ = I: [[2,3],[1,4]]·(1/5)[[4,−3],[−1,2]] = I ✓

  • B) Verify: f(f⁻¹(x)) = 5·(x+3)/5−3 = x+3−3 = x ✓
  • C) y = 5x−3; swap x and y: x = 5y−3

C is correct. y = 5x−3; swap x and y: x = 5y−3

  • C) y = (3x+1)/(x−2); swap: x = (3y+1)/(y−2)
