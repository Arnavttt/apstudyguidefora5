Sensation and Perception

Learn on this site
Sensory Thresholds and Signal Detection

The absolute threshold is the minimum stimulus intensity detectable 50% of the time; the just noticeable difference (JND) is the minimum change detectable; signal detection theory holds that detecting a signal depends on both sensitivity and response criterion (willingness to report).

Common misconception: Subliminal stimuli are below absolute threshold—they can influence behavior slightly but do not have the powerful persuasive effects popular media claims.

How it can be tested: MCQ tests threshold definitions; FRQ applies signal detection theory to real scenarios.

Perceptual Principles: Gestalt and Depth Cues

Gestalt principles (figure-ground, proximity, similarity, closure, continuity) describe how we organize sensory input into meaningful patterns; binocular depth cues (retinal disparity, convergence) and monocular cues (linear perspective, texture gradient) help us perceive 3D space.

Common misconception: Gestalt principles are descriptive, not explanatory—they describe what we perceive but don't fully explain the neural mechanisms.

How it can be tested: MCQ shows visual illusions and asks you to apply Gestalt principles.

Perceptual Set and Top-Down Processing

Top-down processing uses prior knowledge and expectations to interpret sensory input; bottom-up processing builds perception from raw sensory data; perceptual set (context, expectations, motivation) shapes what we perceive.

Common misconception: Top-down processing is not always accurate—it can cause us to see what we expect rather than what is actually there, leading to errors.

How it can be tested: FRQ asks you to apply top-down vs. bottom-up processing to explain perceptual errors.

Key Vocabulary
  • Absolute threshold
  • Just noticeable difference
  • Gestalt principles
  • Top-down processing
  • Perceptual set
Objectives
  • Distinguish absolute threshold from JND using real examples.
  • Apply Gestalt principles to visual organization examples.
  • Explain top-down vs. bottom-up processing and how each can cause perceptual errors.
How this can be tested
  • MCQ tests sensory thresholds and Gestalt principles.
  • FRQ applies signal detection theory.
  • Scenario questions test perceptual set effects.
Practice before moving on
  • List all five Gestalt principles with a sketch illustrating each.
  • Design a signal detection experiment showing how criterion affects detection.

Learning: Classical and Operant Conditioning

Learn on this site
Classical Conditioning

In classical conditioning (Pavlov), a neutral stimulus (NS) becomes a conditioned stimulus (CS) by repeated pairing with an unconditioned stimulus (UCS) that naturally triggers a response (UCR); after conditioning, the CS alone produces the conditioned response (CR).

Common misconception: The CR is not identical to the UCR—it is often weaker; for example, salivation to a bell (CR) is less than salivation to food (UCR).

How it can be tested: FRQ provides a scenario and asks you to identify and label NS, CS, UCS, UCR, and CR.

Operant Conditioning

In operant conditioning (Skinner), behaviors are strengthened by reinforcement (positive: add pleasant stimulus; negative: remove unpleasant stimulus) and weakened by punishment (positive: add aversive stimulus; negative: remove pleasant stimulus).

Common misconception: Negative reinforcement is not punishment—it strengthens behavior by removing something unpleasant (buckling a seatbelt to stop the beeping); students consistently confuse these terms.

How it can be tested: MCQ requires precise distinction between positive/negative reinforcement and positive/negative punishment.

Observational Learning and Cognitive Factors

Observational (social) learning (Bandura) occurs by watching and imitating models; the Bobo doll experiment showed children imitate aggressive behavior they observe; cognitive factors (expectation, self-efficacy) mediate whether observed behaviors are actually performed.

Common misconception: Observational learning does not require reinforcement—children learned aggression from watching even without being rewarded for it themselves.

How it can be tested: FRQ asks you to apply Bandura's research to real-world learning situations.

Key Vocabulary
  • Classical conditioning
  • Operant conditioning
  • Positive reinforcement
  • Negative reinforcement
  • Observational learning
Objectives
  • Label all components of a classical conditioning scenario.
  • Distinguish positive/negative reinforcement from positive/negative punishment with examples.
  • Apply Bandura's observational learning to explain a real-world behavior.
How this can be tested
  • FRQ provides scenario for classical conditioning labeling.
  • MCQ tests precise reinforcement/punishment distinctions.
  • Scenario questions apply learning principles to new situations.
Practice before moving on
  • Create four scenario examples: one for each type of reinforcement and punishment.
  • Research the Bobo doll experiment and identify its key findings and criticisms.

Memory: Encoding, Storage, and Retrieval

Learn on this site
Memory Models and Encoding

The Atkinson-Shiffrin model describes memory flowing from sensory memory → short-term memory (STM, ~7 items, ~30 seconds) → long-term memory (LTM); elaborative encoding (connecting new info to existing knowledge) produces stronger LTM than rote repetition.

Common misconception: Working memory (Baddeley) replaces the passive STM concept—it is an active workspace that manipulates information, not just holds it.

How it can be tested: MCQ tests memory model components; FRQ applies encoding strategies.

Memory Storage and Types

Long-term memory divides into explicit (declarative: episodic/semantic) and implicit (procedural/conditioned) memory; the serial position effect shows better recall for first (primacy) and last (recency) items in a list.

Common misconception: Implicit memories (skills, habits) are not "forgotten"—they are stored differently than explicit memories and can persist even when explicit memory is severely damaged (as in H.M.).

How it can be tested: SAQ asks you to distinguish explicit and implicit memory using patient case studies.

Forgetting and Memory Errors

Forgetting occurs through decay, interference (proactive: old interferes with new; retroactive: new interferes with old), and retrieval failure; memory is reconstructive—the misinformation effect shows false details can be incorporated into memories after the fact.

Common misconception: Memory is not like a video recording—it is actively reconstructed each time we recall it, making it vulnerable to distortion.

How it can be tested: FRQ asks you to explain eyewitness testimony unreliability using memory research.

Key Vocabulary
  • Short-term memory
  • Long-term memory
  • Explicit memory
  • Implicit memory
  • Misinformation effect
Objectives
  • Trace information through the Atkinson-Shiffrin model.
  • Distinguish explicit and implicit memory with examples.
  • Explain the misinformation effect and its implications for eyewitness testimony.
How this can be tested
  • MCQ tests memory model stages.
  • FRQ applies misinformation effect to eyewitness testimony.
  • Case study questions use H.M. to test memory type distinctions.
Practice before moving on
  • Research the misinformation effect (Loftus) and describe one study.
  • Explain why elaborative encoding produces better LTM than maintenance rehearsal.

Thinking, Language, and Intelligence

Learn on this site
Problem Solving and Cognitive Biases

Problem-solving strategies include algorithms (step-by-step guarantees), heuristics (mental shortcuts—faster but error-prone), and insight; cognitive biases include confirmation bias (seeking confirming evidence), availability heuristic (judging probability by ease of recall), and representativeness heuristic.

Common misconception: Heuristics are not irrational—they are efficient shortcuts that usually work; they fail in specific predictable ways that can be identified and corrected.

How it can be tested: MCQ identifies cognitive biases from scenario descriptions.

Language Development and Linguistic Relativity

Language develops from cooing → babbling → one-word (holophrases) → two-word → full syntax; Chomsky proposed a language acquisition device (LAD) enabling universal grammar; the linguistic relativity hypothesis (Whorf) suggests language shapes thought.

Common misconception: The strong version of linguistic relativity (language determines thought) is not supported—people can think about concepts their language lacks words for.

How it can be tested: MCQ tests language development stages; FRQ may ask about nature vs. nurture in language acquisition.

Intelligence: Theories and Measurement

Intelligence theories include Spearman's g factor (general intelligence), Gardner's multiple intelligences, and Sternberg's triarchic theory (analytical, creative, practical); IQ tests measure specific abilities but debate continues about what "intelligence" actually is.

Common misconception: IQ scores are not fixed—they show the Flynn effect (scores have risen about 3 points per decade globally), suggesting environmental factors significantly influence measured intelligence.

How it can be tested: FRQ evaluates different intelligence theories and discusses what IQ tests do and do not measure.

Key Vocabulary
  • Heuristic
  • Confirmation bias
  • Language acquisition device
  • Flynn effect
  • g factor
Objectives
  • Identify and apply cognitive biases to real-world decision-making.
  • Trace language development stages and evaluate nature vs. nurture explanations.
  • Compare intelligence theories and evaluate what IQ tests measure.
How this can be tested
  • MCQ identifies cognitive biases from scenarios.
  • MCQ tests language development stages.
  • FRQ compares intelligence theories.
Practice before moving on
  • Research the Flynn effect and explain two environmental factors that might account for it.
  • Compare Chomsky's nativist and Skinner's behaviorist explanations for language acquisition.
