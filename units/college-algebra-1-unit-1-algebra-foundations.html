Real Numbers, Properties, and Exponent Rules

Learn on this site
Real Number Properties

The real numbers satisfy commutative (a+b=b+a), associative ((a+b)+c=a+(b+c)), distributive (a(b+c)=ab+ac), identity (a+0=a, a·1=a), and inverse (a+(-a)=0) properties. Order of operations: PEMDAS (Parentheses, Exponents, Multiplication/Division left-to-right, Addition/Subtraction left-to-right).

Common misconception: Students forget the distributive property when subtracting: a-(b+c) = a-b-c, not a-b+c.

How it can be tested: Apply properties to simplify expressions; identify violations of properties; evaluate expressions using PEMDAS.

Exponent Rules and Radicals

Key exponent rules: x^a · x^b = x^(a+b), x^a/x^b = x^(a-b), (x^a)^b = x^(ab), x^0 = 1 (x≠0), x^(-n) = 1/x^n. Radicals: √(x^2) = |x|; √a · √b = √(ab); rational exponents: x^(m/n) = (x^(1/n))^m = (ⁿ√x)^m.

Common misconception: Students think √(a²+b²) = a+b (incorrect; you cannot distribute the radical over addition).

How it can be tested: Apply exponent rules to simplify; convert between radical and rational exponent forms; evaluate numerical expressions with exponents.

Key Vocabulary
  • distributive property — a(b + c) = ab + ac; used to expand or factor expressions
  • PEMDAS — Order of operations: Parentheses, Exponents, Multiplication/Division, Addition/Subtraction
  • rational exponent — An exponent that is a fraction: x^(m/n) = the nth root of x, raised to the m power
  • radical — An expression using a root symbol (√); √a means the principal (non-negative) square root of a
  • absolute value — The distance of a number from zero; |x| ≥ 0 always
Objectives
  • Apply the distributive property to expand and factor
  • Simplify expressions using exponent rules
  • Convert between radical and rational exponent form
How this can be tested
  • Simplify 3(x - 2) + 5x using the distributive and combining-like-terms
  • Evaluate x^(-2) · x^5 using exponent rules
Practice before moving on

Q1
The product rule for exponents states that x^3 · x^4 equals
  • A) x^1 (subtract exponents)
  • B) x^7 — add the exponents when multiplying like bases
  • C) x^12 (multiply exponents)
  • D) x^(3/4) (divide exponents)
B is correct. x^7 — add the exponents when multiplying like bases
Q2
The expression (2x^3y)^2 simplifies to
  • A) 2x^6y^2
  • B) 4x^6y
  • C) 2x^5y^2
  • D) 4x^6y^2 — square each factor including the coefficient
D is correct. 4x^6y^2 — square each factor including the coefficient
Q3
The expression x^(-3) is equivalent to
  • A) 1/x^3 — negative exponents indicate reciprocals
  • B) −x^3
  • C) x^(1/3)
  • D) −1/x^3
A is correct. 1/x^3 — negative exponents indicate reciprocals
Q4
√(49x^4) simplifies to
  • A) 7x^2 — take the square root of each factor (assuming x ≥ 0)
  • B) 7x^2 only when x > 0
  • C) 7|x^2|
  • D) 49x^2
A is correct. 7x^2 — take the square root of each factor (assuming x ≥ 0)
Q5
The number 16^(3/4) equals
  • A) 4
  • B) 12
  • C) 64
  • D) 8 — (⁴√16)^3 = 2^3 = 8
D is correct. 8 — (⁴√16)^3 = 2^3 = 8
