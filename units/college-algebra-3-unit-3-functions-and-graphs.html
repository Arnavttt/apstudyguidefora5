Functions: Definition, Graphs, and Transformations

Learn on this site
Functions, Domain, and Range

A function assigns exactly one output to each input. The domain is the set of all valid inputs (x-values); the range is the set of all outputs (y-values). Common domain restrictions: denominators ≠ 0; radicands of even roots ≥ 0. The vertical line test: if any vertical line crosses a graph more than once, the relation is NOT a function.

Common misconception: Students confuse "not a function" with "has no value" — a graph can still have y-values at every x but fail the vertical line test (e.g., a circle).

How it can be tested: Identify functions using the vertical line test; determine domain and range; state domain restrictions for rational/radical functions.

Transformations and Inverse Functions

Graph transformations: f(x)+k shifts up k; f(x+h) shifts left h; -f(x) reflects over x-axis; f(-x) reflects over y-axis; af(x) vertical stretch (a>1) or compression (0

Common misconception: Students confuse horizontal shift direction: f(x+3) shifts LEFT 3 (not right), because x+3=0 when x=-3.

How it can be tested: Apply transformations to sketch graphs; find inverse functions algebraically; use the horizontal line test.

Key Vocabulary
  • domain — The set of all valid input (x) values for a function
  • range — The set of all output (y) values produced by a function
  • vertical line test — A graph represents a function if and only if no vertical line crosses the graph more than once
  • horizontal line test — A function is one-to-one (has an inverse function) if and only if no horizontal line crosses the graph more than once
  • piecewise function — A function defined by different formulas on different intervals of the domain
  • even function — f(-x) = f(x) — symmetric about the y-axis; odd function: f(-x) = -f(x), symmetric about the origin
Objectives
  • Apply the vertical line test to identify functions
  • State the domain and range of common functions
  • Apply transformations and sketch the resulting graph
  • Find inverse functions algebraically
How this can be tested
  • Find the domain of f(x) = √(x-3) / (x-5)
  • If f(x) = 2x+1, find f^(-1)(x) and verify f(f^(-1)(x)) = x

Q1
The domain of f(x) = 1/(x^2 - 9) is
  • A) all real numbers
  • B) x > 0 only
  • C) x ≠ 0
  • D) all real numbers except x = ±3 (where the denominator is zero)
D is correct. all real numbers except x = ±3 (where the denominator is zero)
Q2
The graph of f(x) + 3 is the graph of f(x) shifted
  • A) up 3 units (vertical shift)
  • B) right 3 units
  • C) left 3 units
  • D) down 3 units
A is correct. up 3 units (vertical shift)
Q3
The graph of f(x - 2) is the graph of f(x) shifted
  • A) left 2 units
  • B) up 2 units
  • C) down 2 units
  • D) right 2 units — replace x with x-2 means the graph moves in the positive x direction
D is correct. right 2 units — replace x with x-2 means the graph moves in the positive x direction
Q4
The inverse of f(x) = 3x - 6 is
  • A) f^(-1)(x) = 3x + 6
  • B) f^(-1)(x) = 1/(3x-6)
  • C) f^(-1)(x) = -3x + 6
  • D) f^(-1)(x) = (x + 6)/3 — swap x and y: x = 3y-6, solve for y
D is correct. f^(-1)(x) = (x + 6)/3 — swap x and y: x = 3y-6, solve for y
Q5
(f ∘ g)(x) where f(x) = x^2 and g(x) = x+1 equals
  • A) (x^2)+1
  • B) (x+1)^2 — apply g first (x+1), then apply f (square it)
  • C) x^2 + 1
  • D) x^2(x+1)
B is correct. (x+1)^2 — apply g first (x+1), then apply f (square it)
