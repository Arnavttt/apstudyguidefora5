Graphing and Transforming Trig Functions

Learn on this site
Graphs of Sine and Cosine

f(x) = A sin(Bx - C) + D has amplitude |A|, period 2π/|B|, phase shift C/B (right if positive), and vertical shift D. The sine graph starts at (0,0) going up; cosine starts at maximum (0,1). Key features: 5 special points per period (start, 1/4, 1/2, 3/4, full cycle). Negative A flips the graph over the midline.

Common misconception: Students think the phase shift is -C when it is actually C/B — you must factor B out of the argument first: A sin(B(x - C/B)) + D.

How it can be tested: Identify A, B, C, D in a trig equation; calculate amplitude, period, phase shift, and vertical shift; sketch one complete cycle.

Graphs of Tangent, Cotangent, Secant, and Cosecant

The tangent function y = tan x has period π (not 2π), no amplitude, and vertical asymptotes where cos x = 0 (at x = π/2 + nπ). The cotangent has asymptotes where sin x = 0. Secant y = sec x = 1/cos x has the same asymptotes as tangent and is the reciprocal of cosine. Cosecant y = csc x = 1/sin x mirrors its relationship with sine.

Common misconception: Students apply the amplitude formula to tan, cot, sec, csc — these functions have no finite amplitude (they extend to ±∞).

How it can be tested: Graph tangent and cotangent using asymptotes; graph secant and cosecant as reciprocals of cosine and sine; identify periods and asymptote positions.

Key Vocabulary
  • amplitude — Half the vertical distance between maximum and minimum values; equals |A| in y = A sin(Bx); undefined for tan, cot, sec, csc
  • period — The horizontal length of one complete cycle; for sin/cos: 2π/|B|; for tan/cot: π/|B|
  • phase shift — Horizontal shift of a trig graph; for y = A sin(Bx - C), the phase shift is C/B to the right
  • vertical shift — The value D in y = A sin(Bx) + D; shifts the midline from y=0 to y=D
  • midline — The horizontal line y = D around which the sine or cosine wave oscillates
  • asymptote — For tangent and secant: vertical lines where the function is undefined, at x = π/2 + nπ
Objectives
  • Calculate amplitude, period, phase shift, and vertical shift from an equation
  • Sketch one complete cycle of a transformed trig function
  • Identify asymptotes for tangent and secant functions
How this can be tested
  • Write the equation of a sine function with amplitude 3, period π, phase shift π/4 right, and midline y=2
  • Sketch y = 2cos(3x - π/2) over one complete cycle
Practice before moving on

Q1
The amplitude of f(x) = -4 sin(x) is
  • A) 4π
  • B) 1
  • C) 4 — amplitude is |A| = |-4| = 4; the negative reflects the graph but does not change amplitude
  • D) -4
C is correct. 4 — amplitude is |A| = |-4| = 4; the negative reflects the graph but does not change amplitude
Q2
The period of f(x) = cos(3x) is
  • A) 3π
  • B) 6π
  • C) 2π
  • D) 2π/3 — the period is 2π divided by the coefficient of x
D is correct. 2π/3 — the period is 2π divided by the coefficient of x
Q3
The phase shift of f(x) = sin(2x - π/2) is
  • A) π/2 to the left
  • B) π/4 to the left
  • C) π/4 to the right — phase shift = C/B = (π/2)/2 = π/4
  • D) π/2 to the right
C is correct. π/4 to the right — phase shift = C/B = (π/2)/2 = π/4
Q4
The tangent function has period
  • A) π/2
  • B) 4π
  • C) π — it completes one full pattern every π units, not every 2π units
  • D) 2π
C is correct. π — it completes one full pattern every π units, not every 2π units
Q5
The vertical asymptotes of y = tan(x) occur at
  • A) x = π/2 + nπ for integer n — where cosine (the denominator) equals zero
  • B) x = nπ (multiples of π)
  • C) x = π/4 + nπ/2
  • D) x = 0
A is correct. x = π/2 + nπ for integer n — where cosine (the denominator) equals zero
