Fundamental Identities

The fundamental identities: Pythagorean (sin^2θ+cos^2θ=1, derived forms: tan^2θ+1=sec^2θ, 1+cot^2θ=csc^2θ), reciprocal (cscθ=1/sinθ, secθ=1/cosθ, cotθ=1/tanθ), quotient (tanθ=sinθ/cosθ, cotθ=cosθ/sinθ), and even/odd (cos(-θ)=cosθ even; sin(-θ)=-sinθ odd). To prove an identity: work on one side only, transforming it algebraically until it matches the other.

Common misconception: Students try to "cross multiply" or move terms across the equals sign when proving identities — you must work one side only.

How it can be tested: Simplify trig expressions using fundamental identities; prove trig identities by transforming one side; apply Pythagorean identities.

Sum, Difference, Double-Angle, and Half-Angle Formulas

Sum/difference: sin(A±B) = sinAcosB ± cosAsinB; cos(A±B) = cosAcosB ∓ sinAsinB. Double-angle: sin(2θ) = 2sinθcosθ; cos(2θ) = cos^2θ-sin^2θ = 2cos^2θ-1 = 1-2sin^2θ. Half-angle: sin(θ/2) = ±√[(1-cosθ)/2]; cos(θ/2) = ±√[(1+cosθ)/2]. These formulas allow computation of exact values beyond the standard special angles.

Common misconception: Students think cos(A+B) = cosA+cosB — the sum formula is NOT additive; you must use the full formula with the cross terms.

How it can be tested: Apply sum/difference formulas to find exact values; use double-angle formulas to simplify; solve equations using half-angle substitutions.

Key Vocabulary
  • Pythagorean identity — sin^2θ + cos^2θ = 1; derived from the unit circle equation x^2+y^2=1
  • double-angle formula — Formulas for sin(2θ) and cos(2θ) in terms of sinθ and cosθ; derived from sum formulas with A=B=θ
  • half-angle formula — sin(θ/2) = ±√[(1-cosθ)/2]; cos(θ/2) = ±√[(1+cosθ)/2]; the sign depends on the quadrant of θ/2
  • co-function identity — sin(π/2-θ) = cosθ; cos(π/2-θ) = sinθ — complementary angles have swapped sin and cos
  • even function — f(-x) = f(x); cosine is even: cos(-θ) = cosθ
  • odd function — f(-x) = -f(x); sine is odd: sin(-θ) = -sinθ

  • A) sinAcosB + cosAsinB (the correct formula)

  • D) sin A + sin B — you must use the full formula: sinAcosB + cosAsinB

Q3
The double-angle formula sin(2θ) equals
  • A) 2sin^2θ
  • B) sin^2θ - cos^2θ
  • C) 2sinθcosθ — derived from sin(θ+θ) = sinθcosθ + cosθsinθ
  • D) sin^2θ + cos^2θ = 1
C is correct. 2sinθcosθ — derived from sin(θ+θ) = sinθcosθ + cosθsinθ
Q4
The exact value of cos(15°) = cos(45°-30°) requires
  • A) the difference formula: cos45°cos30° + sin45°sin30°
  • B) the half-angle formula: cos(30°/2)
  • C) the Pythagorean identity only
  • D) the double-angle formula: cos(30°/2)
A is correct. the difference formula: cos45°cos30° + sin45°sin30°
Q5
The co-function identity cos(π/2 - θ) equals
  • A) -cos θ
  • B) sin θ — cosine of the complement equals sine of the angle
  • C) cos θ
  • D) -sin θ
B is correct. sin θ — cosine of the complement equals sine of the angle
