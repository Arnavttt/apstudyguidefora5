Inverse Trig Functions and Solving Trig Equations

Learn on this site
Inverse Trigonometric Functions

Since trig functions are periodic (not one-to-one), their inverses require restricted domains. arcsin: domain [-1,1], range [-π/2, π/2]. arccos: domain [-1,1], range [0, π]. arctan: domain (-∞,∞), range (-π/2, π/2). Composition: sin(arcsin x) = x for x ∈ [-1,1]; arcsin(sin x) = x only for x ∈ [-π/2, π/2] (principal range).

Common misconception: Students think arcsin(sin 200°) = 200° — arcsin only returns values in [-90°, 90°], so arcsin(sin 200°) = -20° (the equivalent angle in the principal range).

How it can be tested: State domains and ranges of inverse trig functions; evaluate compositions; find principal values.

Solving Trigonometric Equations

To solve sin x = k: find the reference angle (arcsin|k|), then place the solution in the correct quadrant(s). On [0, 2π): sine positive in Q1 and Q2; cosine positive in Q1 and Q4; tangent positive in Q1 and Q3. General solutions add ±2πn (or πn for tangent). For equations requiring algebra first (e.g., 2sin^2x - sinx - 1 = 0), factor or use substitution u = sinx.

Common misconception: Students find one solution and stop — always find ALL solutions on the given interval, and use general form (±2πn) when the domain is unrestricted.

How it can be tested: Find all solutions to trig equations on [0, 2π); write general solutions; solve trig equations requiring factoring or substitution.

Key Vocabulary
  • arcsin (sin^(-1)) — The inverse of sin with restricted domain: takes input in [-1,1] and returns angle in [-π/2, π/2]
  • arccos (cos^(-1)) — The inverse of cos with restricted domain: returns angle in [0, π]
  • arctan (tan^(-1)) — The inverse of tan: returns angle in (-π/2, π/2)
  • principal value — The unique output of an inverse trig function within its restricted range
  • general solution — All solutions to a trig equation: for sin/cos add ±2πn; for tan add πn
  • extraneous solution — A value that satisfies a derived equation but not the original; check domain after squaring or substituting
